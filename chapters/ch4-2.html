บทที่ 4.2: สมการกำลังสองตัวแปรเดียว

ศึกษาสมการกำลังสองตัวแปรเดียว ซึ่งเป็นสมการที่ปรากฏบ่อยครั้งในโจทย์คณิตศาสตร์และวิทยาการคำนวณ โดยเฉพาะในเรื่องที่เกี่ยวข้องกับการหาค่าสูงสุด-ต่ำสุด การเคลื่อนที่ หรือพื้นที่ต่างๆ

เนื้อหาในบทเรียน

  • 4.2.1 ความหมายและรูปแบบของสมการกำลังสองตัวแปรเดียว

  • 4.2.4 ตัวอย่างโจทย์และแนวทางการทำข้อสอบ

4.2.1 ความหมายและรูปแบบของสมการกำลังสองตัวแปรเดียว

นิยาม: สมการกำลังสองตัวแปรเดียว (Quadratic Equation in One Variable)

สมการกำลังสองตัวแปรเดียว คือ สมการที่สามารถจัดรูปได้เป็น:

$$ax^2 + bx + c = 0$$

เมื่อ:

  • $x$ คือ ตัวแปร
  • $a, b, c$ คือ ค่าคงที่ใดๆ ที่เป็นจำนวนจริง
  • เงื่อนไขสำคัญคือ $a \neq 0$ (ถ้า $a = 0$ สมการจะกลายเป็นสมการเชิงเส้น)

ตัวอย่างสมการกำลังสองตัวแปรเดียว

  • $x^2 + 5x + 6 = 0$
  • $2y^2 - 7 = 0$
  • $3z^2 - 4z = 0$
  • $(x - 1)^2 = 4$

4.2.2 วิธีการแก้สมการกำลังสอง (Methods for Solving Quadratic Equations)

การแก้สมการกำลังสองคือการหาค่าของตัวแปร $x$ ที่ทำให้สมการเป็นจริง ซึ่งค่าเหล่านี้เรียกว่า รากของสมการ (roots) หรือ คำตอบของสมการ (solutions) มีหลายวิธีดังนี้:

ก. การแยกตัวประกอบ (Factoring)

วิธีนี้ใช้ได้เมื่อพหุนาม $ax^2 + bx + c$ สามารถแยกตัวประกอบได้ง่าย

ขั้นตอน:

  1. จัดสมการให้อยู่ในรูป $ax^2 + bx + c = 0$
  2. แยกตัวประกอบของพหุนาม $ax^2 + bx + c$ ให้อยู่ในรูปผลคูณของพหุนามเชิงเส้นสองตัว เช่น $(px + q)(rx + s) = 0$
  3. กำหนดให้แต่ละตัวประกอบเท่ากับศูนย์ แล้วแก้สมการหาค่า $x$

ตัวอย่าง

โจทย์: จงแก้สมการ $x^2 - 5x + 6 = 0$

วิธีทำ:

  1. แยกตัวประกอบ: $(x - 2)(x - 3) = 0$
  2. กำหนดให้แต่ละตัวประกอบเท่ากับศูนย์:
    • $x - 2 = 0 \Rightarrow x = 2$
    • $x - 3 = 0 \Rightarrow x = 3$

คำตอบ: $x = 2$ หรือ $x = 3$

ข. การใช้สูตร (Using the Quadratic Formula)

วิธีนี้ใช้ได้กับสมการกำลังสองทุกรูปแบบ และเป็นวิธีที่แนะนำเมื่อไม่สามารถแยกตัวประกอบได้ง่ายๆ

สูตรกำลังสอง:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

ขั้นตอน:

  1. จัดสมการให้อยู่ในรูป $ax^2 + bx + c = 0$
  2. ระบุค่า $a, b, c$
  3. แทนค่า $a, b, c$ ลงในสูตรแล้วคำนวณหาค่า $x$

ตัวอย่าง

โจทย์: จงแก้สมการ $2x^2 + 3x - 2 = 0$

วิธีทำ:

  1. สมการอยู่ในรูป $ax^2 + bx + c = 0$ แล้ว โดย $a = 2, b = 3, c = -2$
  2. แทนค่าในสูตร: $$x = \frac{-3 \pm \sqrt{3^2 - 4(2)(-2)}}{2(2)}$$ $$x = \frac{-3 \pm \sqrt{9 - (-16)}}{4}$$ $$x = \frac{-3 \pm \sqrt{9 + 16}}{4}$$ $$x = \frac{-3 \pm \sqrt{25}}{4}$$ $$x = \frac{-3 \pm 5}{4}$$
  3. แยกคำตอบ:
    • $x_1 = \frac{-3 + 5}{4} = \frac{2}{4} = \frac{1}{2}$
    • $x_2 = \frac{-3 - 5}{4} = \frac{-8}{4} = -2$

คำตอบ: $x = \frac{1}{2}$ หรือ $x = -2$

ค. การทำให้เป็นกำลังสองสมบูรณ์ (Completing the Square)

วิธีนี้ใช้เป็นพื้นฐานในการพิสูจน์ที่มาของสูตรกำลังสอง และบางครั้งก็ใช้แก้สมการได้โดยตรง

แนวคิด: จัดรูปสมการให้อยู่ในรูป $(x + k)^2 = D$ หรือ $(x - k)^2 = D$

4.2.3 ลักษณะของคำตอบของสมการกำลังสอง (Nature of Roots - Discriminant)

ค่าที่อยู่ใต้เครื่องหมายกรณฑ์ในสูตรกำลังสองคือ $b^2 - 4ac$ ซึ่งเรียกว่า ดิสคริมิแนนต์ (Discriminant) เขียนแทนด้วย $\Delta$ หรือ $D$

ค่าของ $D$ เป็นตัวบอกลักษณะของคำตอบของสมการกำลังสอง:

$$D = b^2 - 4ac$$
  • ถ้า $D > 0$ $(b^2 - 4ac > 0)$: สมการมีคำตอบเป็น จำนวนจริง 2 คำตอบที่แตกต่างกัน
  • ถ้า $D = 0$ $(b^2 - 4ac = 0)$: สมการมีคำตอบเป็น จำนวนจริง 1 คำตอบ (หรือ 2 คำตอบที่เท่ากัน/ซ้ำกัน)
  • ถ้า $D < 0$ $(b^2 - 4ac < 0)$: สมการ ไม่มีคำตอบเป็นจำนวนจริง (คำตอบเป็นจำนวนเชิงซ้อน 2 คำตอบที่แตกต่างกัน)

ตัวอย่างการใช้ดิสคริมิแนนต์

พิจารณาสมการ $x^2 - 6x + 9 = 0$

$a = 1, b = -6, c = 9$

$D = (-6)^2 - 4(1)(9) = 36 - 36 = 0$

เนื่องจาก $D = 0$ ดังนั้น สมการมีคำตอบเป็นจำนวนจริง 1 คำตอบ (รากซ้ำ)

4.2.4 ตัวอย่างโจทย์และแนวทางการทำข้อสอบ

โจทย์ข้อที่ 1: การแก้สมการโดยการแยกตัวประกอบ

คำถาม: จงหาคำตอบของสมการ $x^2 + 7x + 10 = 0$

แนวคิด:

  • แยกตัวประกอบของพหุนาม
  • กำหนดให้แต่ละตัวประกอบเท่ากับศูนย์

เฉลย:

$x^2 + 7x + 10 = 0$

แยกตัวประกอบ: $(x + 2)(x + 5) = 0$

$x + 2 = 0 \Rightarrow x = -2$

$x + 5 = 0 \Rightarrow x = -5$

คำตอบ: $x = -2$ หรือ $x = -5$

โจทย์ข้อที่ 2: การแก้สมการโดยใช้สูตรกำลังสอง

คำถาม: จงหาคำตอบของสมการ $3x^2 - 4x - 1 = 0$

แนวคิด:

  • ระบุค่า $a, b, c$
  • แทนค่าในสูตร $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

เฉลย:

จากสมการ $3x^2 - 4x - 1 = 0$, เราได้ $a = 3, b = -4, c = -1$

แทนค่าในสูตร:

$$x = \frac{-(-4) \pm \sqrt{(-4)^2 - 4(3)(-1)}}{2(3)}$$ $$x = \frac{4 \pm \sqrt{16 + 12}}{6}$$ $$x = \frac{4 \pm \sqrt{28}}{6}$$ $$x = \frac{4 \pm 2\sqrt{7}}{6}$$ $$x = \frac{2(2 \pm \sqrt{7})}{6}$$ $$x = \frac{2 \pm \sqrt{7}}{3}$$

คำตอบ: $x = \frac{2 + \sqrt{7}}{3}$ หรือ $x = \frac{2 - \sqrt{7}}{3}$

โจทย์ข้อที่ 3: การจัดรูปสมการก่อนแก้

คำถาม: จงหาคำตอบของสมการ $x(x - 4) = 5$

แนวคิด:

  • กระจายพหุนามทางซ้ายมือ
  • จัดสมการให้อยู่ในรูป $ax^2 + bx + c = 0$
  • แยกตัวประกอบหรือใช้สูตร

เฉลย:

$x(x - 4) = 5$

$x^2 - 4x = 5$

$x^2 - 4x - 5 = 0$

แยกตัวประกอบ: $(x - 5)(x + 1) = 0$

$x - 5 = 0 \Rightarrow x = 5$

$x + 1 = 0 \Rightarrow x = -1$

คำตอบ: $x = 5$ หรือ $x = -1$

โจทย์ข้อที่ 4: การพิจารณาลักษณะคำตอบ (ดิสคริมิแนนต์)

คำถาม: จงพิจารณาว่าสมการ $x^2 - 6x + 9 = 0$ มีคำตอบเป็นจำนวนจริงกี่คำตอบ และเป็นอย่างไร?

แนวคิด:

  • ระบุค่า $a, b, c$
  • คำนวณค่าดิสคริมิแนนต์ $D = b^2 - 4ac$
  • ตีความค่า $D$

เฉลย:

จากสมการ $x^2 - 6x + 9 = 0$, เราได้ $a = 1, b = -6, c = 9$

คำนวณดิสคริมิแนนต์:

$D = b^2 - 4ac = (-6)^2 - 4(1)(9)$

$D = 36 - 36$

$D = 0$

เนื่องจาก $D = 0$ สมการมีคำตอบเป็น จำนวนจริง 1 คำตอบ (หรือ 2 คำตอบที่เท่ากัน)

คำตอบ: มีคำตอบเป็นจำนวนจริง 1 คำตอบ (ซึ่งเป็นคำตอบซ้ำ)

โจทย์ข้อที่ 5: โจทย์ปัญหาประยุกต์

คำถาม: รูปสี่เหลี่ยมผืนผ้ามีพื้นที่ 48 ตารางหน่วย โดยมีด้านยาวยาวกว่าด้านกว้างอยู่ 2 หน่วย จงหาความยาวของด้านแต่ละด้านของรูปสี่เหลี่ยมผืนผ้า

แนวคิด:

  • กำหนดตัวแปรแทนความยาวด้านกว้าง (หรือด้านยาว)
  • เขียนความสัมพันธ์ของด้านยาวกับด้านกว้าง
  • ใช้สูตรพื้นที่สี่เหลี่ยมผืนผ้า (กว้าง × ยาว) เพื่อสร้างสมการกำลังสอง
  • แก้สมการกำลังสอง
  • เลือกคำตอบที่สมเหตุสมผล (ความยาวต้องเป็นบวก)

เฉลย:

ให้ด้านกว้างของรูปสี่เหลี่ยมผืนผ้าเป็น $x$ หน่วย

ดังนั้น ด้านยาวจะเป็น $x + 2$ หน่วย

พื้นที่สี่เหลี่ยมผืนผ้า = กว้าง × ยาว

$48 = x(x + 2)$

$48 = x^2 + 2x$

$x^2 + 2x - 48 = 0$

แยกตัวประกอบ: $(x + 8)(x - 6) = 0$

$x + 8 = 0 \Rightarrow x = -8$

$x - 6 = 0 \Rightarrow x = 6$

เนื่องจากความยาวต้องเป็นค่าบวก เราจึงเลือก $x = 6$

ด้านกว้าง = 6 หน่วย

ด้านยาว = $x + 2 = 6 + 2 = 8$ หน่วย

คำตอบ: ด้านกว้าง 6 หน่วย, ด้านยาว 8 หน่วย

4.2.5 ตัวอย่างข้อสอบฝึกทำ

ข้อที่ 1

จงหาคำตอบของสมการ $x^2 + 8x + 15 = 0$

แนวคิด

แยกตัวประกอบของพหุนาม

กำหนดให้แต่ละตัวประกอบเท่ากับศูนย์

เฉลย

$x^2 + 8x + 15 = 0$

แยกตัวประกอบ: $(x + 3)(x + 5) = 0$

$x + 3 = 0 \Rightarrow x = -3$

$x + 5 = 0 \Rightarrow x = -5$

คำตอบ: $x = -3$ หรือ $x = -5$

ข้อที่ 2

จงหาคำตอบของสมการ $2x^2 + 5x - 3 = 0$

แนวคิด

ระบุค่า $a, b, c$

แทนค่าในสูตร $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

เฉลย

จากสมการ $2x^2 + 5x - 3 = 0$, เราได้ $a = 2, b = 5, c = -3$

แทนค่าในสูตร:

$x = \frac{-5 \pm \sqrt{5^2 - 4(2)(-3)}}{2(2)}$

$x = \frac{-5 \pm \sqrt{25 - (-24)}}{4}$

$x = \frac{-5 \pm \sqrt{25 + 24}}{4}$

$x = \frac{-5 \pm \sqrt{49}}{4}$

$x = \frac{-5 \pm 7}{4}$

แยกคำตอบ:

  • $x_1 = \frac{-5 + 7}{4} = \frac{2}{4} = \frac{1}{2}$
  • $x_2 = \frac{-5 - 7}{4} = \frac{-12}{4} = -3$

คำตอบ: $x = \frac{1}{2}$ หรือ $x = -3$

ข้อที่ 3

จงหาคำตอบของสมการ $x(x + 6) = 7$

แนวคิด

กระจายพหุนามทางซ้ายมือ

จัดสมการให้อยู่ในรูป $ax^2 + bx + c = 0$

แยกตัวประกอบหรือใช้สูตร

เฉลย

$x(x + 6) = 7$

$x^2 + 6x = 7$

$x^2 + 6x - 7 = 0$

แยกตัวประกอบ: $(x + 7)(x - 1) = 0$

$x + 7 = 0 \Rightarrow x = -7$

$x - 1 = 0 \Rightarrow x = 1$

คำตอบ: $x = -7$ หรือ $x = 1$

ข้อที่ 4

จงหาคำตอบของสมการ $3x^2 - 27 = 0$

แนวคิด

ย้ายพจน์คงที่ไปอีกข้าง

หารด้วยสัมประสิทธิ์หน้า $x^2$

ถอดรากที่สองทั้งสองข้าง (ระวังค่าบวกและลบ)

เฉลย

$3x^2 - 27 = 0$

$3x^2 = 27$

$x^2 = \frac{27}{3}$

$x^2 = 9$

ถอดรากที่สอง: $x = \pm\sqrt{9}$

$x = \pm 3$

คำตอบ: $x = 3$ หรือ $x = -3$

ข้อที่ 5

จงพิจารณาว่าสมการ $x^2 + 2x + 5 = 0$ มีคำตอบเป็นจำนวนจริงกี่คำตอบ และเป็นอย่างไร?

แนวคิด

ระบุค่า $a, b, c$

คำนวณค่าดิสคริมิแนนต์ $D = b^2 - 4ac$

ตีความค่า $D$:

  • $D > 0$: 2 คำตอบจริงที่แตกต่างกัน
  • $D = 0$: 1 คำตอบจริง (ซ้ำกัน)
  • $D < 0$: ไม่มีคำตอบจริง (คำตอบเชิงซ้อน)

เฉลย

จากสมการ $x^2 + 2x + 5 = 0$, เราได้ $a = 1, b = 2, c = 5$

คำนวณดิสคริมิแนนต์:

$D = b^2 - 4ac = (2)^2 - 4(1)(5)$

$D = 4 - 20$

$D = -16$

เนื่องจาก $D = -16 < 0$ สมการ ไม่มีคำตอบเป็นจำนวนจริง (คำตอบเป็นจำนวนเชิงซ้อน 2 คำตอบ)

คำตอบ: ไม่มีคำตอบเป็นจำนวนจริง

ข้อที่ 6

รูปสี่เหลี่ยมจัตุรัสมีพื้นที่ 121 ตารางเซนติเมตร จงหาความยาวของด้านแต่ละด้านของรูปสี่เหลี่ยมจัตุรัสนี้

แนวคิด

กำหนดตัวแปรแทนความยาวด้าน

ใช้สูตรพื้นที่สี่เหลี่ยมจัตุรัส (ด้าน × ด้าน) เพื่อสร้างสมการ

แก้สมการกำลังสอง (โดยการถอดรากที่สอง)

เลือกคำตอบที่สมเหตุสมผล (ความยาวต้องเป็นบวก)

เฉลย

ให้ด้านของรูปสี่เหลี่ยมจัตุรัสเป็น $x$ เซนติเมตร

พื้นที่สี่เหลี่ยมจัตุรัส = ด้าน × ด้าน

$121 = x^2$

ถอดรากที่สอง: $x = \pm\sqrt{121}$

$x = \pm 11$

เนื่องจากความยาวต้องเป็นค่าบวก เราจึงเลือก $x = 11$

คำตอบ: ความยาวด้านแต่ละด้านคือ 11 เซนติเมตร

ข้อที่ 7

จงหาคำตอบของสมการ $(x + 2)^2 = 3x + 10$

แนวคิด

กระจาย $(x + 2)^2$ (ใช้สูตร $(a + b)^2 = a^2 + 2ab + b^2$)

ย้ายทุกพจน์ไปอยู่ข้างเดียวกันเพื่อจัดให้อยู่ในรูป $ax^2 + bx + c = 0$

แยกตัวประกอบหรือใช้สูตร

เฉลย

$(x + 2)^2 = 3x + 10$

$x^2 + 4x + 4 = 3x + 10$

ย้ายพจน์ทั้งหมดมาอยู่ข้างเดียวกัน:

$x^2 + 4x - 3x + 4 - 10 = 0$

$x^2 + x - 6 = 0$

แยกตัวประกอบ: $(x + 3)(x - 2) = 0$

$x + 3 = 0 \Rightarrow x = -3$

$x - 2 = 0 \Rightarrow x = 2$

คำตอบ: $x = -3$ หรือ $x = 2$

ข้อที่ 8

จงหาค่า $k$ ที่ทำให้สมการ $x^2 - 4x + k = 0$ มีคำตอบเป็นจำนวนจริงเพียง 1 คำตอบ

แนวคิด

เงื่อนไขที่สมการมีคำตอบเป็นจำนวนจริงเพียง 1 คำตอบคือ ดิสคริมิแนนต์ $D = b^2 - 4ac = 0$

ระบุค่า $a, b, c$ แล้วแทนลงในสูตรดิสคริมิแนนต์

แก้สมการหา $k$

เฉลย

จากสมการ $x^2 - 4x + k = 0$, เราได้ $a = 1, b = -4, c = k$

เพื่อให้มีคำตอบเป็นจำนวนจริงเพียง 1 คำตอบ (รากซ้ำ) ดิสคริมิแนนต์ต้องเป็นศูนย์:

$D = b^2 - 4ac = 0$

$(-4)^2 - 4(1)(k) = 0$

$16 - 4k = 0$

$16 = 4k$

$k = \frac{16}{4}$

$k = 4$

คำตอบ: $k = 4$

ข้อที่ 9

ผลคูณของจำนวนเต็มบวกสองจำนวนที่เรียงติดกันคือ 90 จงหาจำนวนทั้งสองนั้น

แนวคิด

กำหนดตัวแปรแทนจำนวนแรก

เขียนนิพจน์แทนจำนวนที่สอง

ตั้งสมการผลคูณ

แก้สมการกำลังสอง

เลือกคำตอบที่สมเหตุสมผล (จำนวนเต็มบวก)

เฉลย

ให้จำนวนเต็มบวกจำนวนแรกเป็น $x$

ดังนั้น จำนวนเต็มบวกที่เรียงติดกันคือ $x + 1$

ผลคูณของจำนวนทั้งสองคือ 90:

$x(x + 1) = 90$

$x^2 + x = 90$

$x^2 + x - 90 = 0$

แยกตัวประกอบ: $(x + 10)(x - 9) = 0$

$x + 10 = 0 \Rightarrow x = -10$

$x - 9 = 0 \Rightarrow x = 9$

เนื่องจากเป็นจำนวนเต็มบวก เราเลือก $x = 9$

จำนวนทั้งสองคือ 9 และ $x + 1 = 9 + 1 = 10$

คำตอบ: 9 และ 10

ข้อที่ 10

จงหาคำตอบของสมการ $(x - 3)(x + 4) = 2x(x - 1)$

แนวคิด

กระจายพหุนามทั้งสองข้างของสมการ

ย้ายทุกพจน์ไปอยู่ข้างเดียวกันเพื่อจัดให้อยู่ในรูป $ax^2 + bx + c = 0$

แก้สมการกำลังสอง (อาจต้องใช้สูตร)

เฉลย

$(x - 3)(x + 4) = 2x(x - 1)$

กระจายพหุนาม:

$x^2 + 4x - 3x - 12 = 2x^2 - 2x$

$x^2 + x - 12 = 2x^2 - 2x$

ย้ายทุกพจน์ไปอยู่ข้างเดียวกัน (นิยมย้ายไปฝั่งที่ทำให้สัมประสิทธิ์หน้า $x^2$ เป็นบวก):

$0 = 2x^2 - x^2 - 2x - x + 12$

$0 = x^2 - 3x + 12$

จากสมการ $x^2 - 3x + 12 = 0$, เราได้ $a = 1, b = -3, c = 12$

ตรวจสอบดิสคริมิแนนต์:

$D = b^2 - 4ac = (-3)^2 - 4(1)(12)$

$D = 9 - 48$

$D = -39$

เนื่องจาก $D = -39 < 0$ สมการ ไม่มีคำตอบเป็นจำนวนจริง

คำตอบ: ไม่มีคำตอบเป็นจำนวนจริง (หรือมีคำตอบเป็นจำนวนเชิงซ้อน)
