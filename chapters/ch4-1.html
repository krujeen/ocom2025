บทที่ 4.1: อสมการเชิงเส้นตัวแปรเดียว

ศึกษาหลักการแก้อสมการเชิงเส้นตัวแปรเดียว ซึ่งเป็นหัวข้อพื้นฐานแต่สำคัญมากในคณิตศาสตร์ และเป็นแนวคิดที่ใช้ในการกำหนดเงื่อนไขต่างๆ ในการเขียนโปรแกรมคอมพิวเตอร์

4.1.1 ความหมายของอสมการเชิงเส้นตัวแปรเดียว

นิยาม: อสมการ (Inequality)

อสมการ คือ ประโยคสัญลักษณ์ที่แสดงความสัมพันธ์ไม่เท่ากันของจำนวน โดยมีเครื่องหมายแสดงความไม่เท่ากัน ได้แก่:

  • $<$ (น้อยกว่า)
  • $>$ (มากกว่า)
  • $\leq$ (น้อยกว่าหรือเท่ากับ)
  • $\geq$ (มากกว่าหรือเท่ากับ)
  • $\neq$ (ไม่เท่ากับ)

นิยาม: อสมการเชิงเส้นตัวแปรเดียว (Linear Inequality in One Variable)

อสมการเชิงเส้นตัวแปรเดียว คือ อสมการที่มีตัวแปรเพียงตัวเดียว และเลขชี้กำลังของตัวแปรนั้นเป็น 1 เท่านั้น โดยสามารถจัดรูปได้เป็น:

$$ax + b < 0$$ $$ax + b > 0$$ $$ax + b \leq 0$$ $$ax + b \geq 0$$ $$ax + b \neq 0$$

เมื่อ $a, b$ เป็นค่าคงที่ใดๆ โดยที่ $a \neq 0$ และ $x$ เป็นตัวแปร

ตัวอย่างอสมการเชิงเส้นตัวแปรเดียว

  • $3x + 5 < 10$
  • $y - 7 \geq 2y$
  • $5(z - 1) \neq 0$
  • $-2x + 3 > 1$
  • $\frac{x}{4} \leq 5$

4.1.2 สมบัติของอสมการ

การแก้อสมการนั้นคล้ายกับการแก้สมการ แต่มีข้อควรระวังบางประการเกี่ยวกับสมบัติการคูณและการหารด้วยจำนวนลบ

ให้ $a, b, c$ เป็นจำนวนจริง

1. สมบัติการบวกด้วยจำนวนที่เท่ากัน

$$\text{ถ้า } a < b \text{ แล้ว } a + c < b + c$$

(ใช้ได้กับทุกเครื่องหมายอสมการ: $>, \leq, \geq$)

ตัวอย่าง: ถ้า $x - 3 < 5$, บวก 3 ทั้งสองข้าง:

$x - 3 + 3 < 5 + 3 \Rightarrow x < 8$

2. สมบัติการคูณด้วยจำนวนบวกที่เท่ากัน

$$\text{ถ้า } a < b \text{ และ } c > 0 \text{ แล้ว } ac < bc$$

(ใช้ได้กับทุกเครื่องหมายอสมการ: $>, \leq, \geq$)

ตัวอย่าง: ถ้า $\frac{x}{2} < 4$, คูณ 2 ทั้งสองข้าง:

$\frac{x}{2} \cdot 2 < 4 \cdot 2 \Rightarrow x < 8$

3. สมบัติการคูณด้วยจำนวนลบที่เท่ากัน

$$\text{ถ้า } a < b \text{ และ } c < 0 \text{ แล้ว } ac > bc \text{ (ต้องเปลี่ยนเครื่องหมายอสมการ)}$$

(ใช้ได้กับทุกเครื่องหมายอสมการ: ถ้า $>$ เป็น $<$, ถ้า $\leq$ เป็น $\geq$ เป็นต้น)

ตัวอย่าง: ถ้า $-3x < 12$, คูณ $-\frac{1}{3}$ ทั้งสองข้าง:

$(-3x) \cdot \left(-\frac{1}{3}\right) > 12 \cdot \left(-\frac{1}{3}\right) \Rightarrow x > -4$ (เปลี่ยนจาก $<$ เป็น $>$)

4. สมบัติการถ่ายทอด

$$\text{ถ้า } a < b \text{ และ } b < c \text{ แล้ว } a < c$$ $$\text{ถ้า } a > b \text{ และ } b > c \text{ แล้ว } a > c$$

4.1.3 การแก้อสมการเชิงเส้นตัวแปรเดียว

การแก้อสมการคือการหาเซตของค่าตัวแปรที่ทำให้อสมการเป็นจริง มีขั้นตอนคล้ายการแก้สมการ:

ขั้นตอนการแก้อสมการ

  1. จัดรูปอสมการให้ตัวแปรอยู่ข้างหนึ่ง และค่าคงที่อีกข้างหนึ่ง
  2. ใช้สมบัติการบวก/ลบ เพื่อย้ายพจน์
  3. ใช้สมบัติการคูณ/หาร เพื่อกำจัดสัมประสิทธิ์หน้าตัวแปร
  4. สำคัญมาก: หากคูณหรือหารด้วยจำนวนลบ ต้อง กลับเครื่องหมายอสมการ

การแสดงคำตอบของอสมการ

1. บนเส้นจำนวน (Number Line)

ใช้จุดทึบ/วงกลมโปร่ง และลากเส้น:

  • $<$ หรือ $>$ : ใช้วงกลมโปร่ง (ไม่รวมค่าที่จุดนั้น)
  • $\leq$ หรือ $\geq$ : ใช้จุดทึบ (รวมค่าที่จุดนั้น)

2. สัญกรณ์ช่วง (Interval Notation)

  • $(a, b)$ : $a < x < b$ (ช่วงเปิด, ไม่รวม $a, b$)
  • $[a, b]$ : $a \leq x \leq b$ (ช่วงปิด, รวม $a, b$)
  • $(a, b]$ : $a < x \leq b$
  • $[a, b)$ : $a \leq x < b$
  • $(a, \infty)$ : $x > a$
  • $[a, \infty)$ : $x \geq a$
  • $(-\infty, a)$ : $x < a$
  • $(-\infty, a]$ : $x \leq a$
  • $(-\infty, \infty)$ : จำนวนจริงทั้งหมด $(\mathbb{R})$

4.1.4 ตัวอย่างโจทย์และแนวทางการทำข้อสอบ

โจทย์ข้อที่ 1: การแก้อสมการพื้นฐาน

คำถาม: จงแก้อสมการ $2x - 7 < 3$ และแสดงคำตอบในรูปสัญกรณ์ช่วงและบนเส้นจำนวน

แนวคิด:

  • ย้ายพจน์ที่เป็นค่าคงที่ไปอีกฝั่ง
  • กำจัดสัมประสิทธิ์หน้าตัวแปร
  • แสดงคำตอบ

เฉลย:

$2x - 7 < 3$

บวก 7 ทั้งสองข้าง:

$2x - 7 + 7 < 3 + 7$

$2x < 10$

หารด้วย 2 (จำนวนบวก):

$\frac{2x}{2} < \frac{10}{2}$

$x < 5$

สัญกรณ์ช่วง: $(-\infty, 5)$

บนเส้นจำนวน: วงกลมโปร่งที่ 5 และลากเส้นไปทางซ้าย

โจทย์ข้อที่ 2: การแก้อสมการที่มีการคูณด้วยจำนวนลบ

คำถาม: จงแก้อสมการ $-4x + 1 \geq 13$ และแสดงคำตอบในรูปสัญกรณ์ช่วงและบนเส้นจำนวน

แนวคิด:

  • ย้ายพจน์ที่เป็นค่าคงที่ไปอีกฝั่ง
  • กำจัดสัมประสิทธิ์หน้าตัวแปร (ระวังเรื่องการกลับเครื่องหมาย)
  • แสดงคำตอบ

เฉลย:

$-4x + 1 \geq 13$

ลบ 1 ทั้งสองข้าง:

$-4x + 1 - 1 \geq 13 - 1$

$-4x \geq 12$

หารด้วย -4 (ต้องกลับเครื่องหมายอสมการ):

$\frac{-4x}{-4} \leq \frac{12}{-4}$

$x \leq -3$

สัญกรณ์ช่วง: $(-\infty, -3]$

บนเส้นจำนวน: จุดทึบที่ -3 และลากเส้นไปทางซ้าย

โจทย์ข้อที่ 3: การแก้อสมการที่มีตัวแปรทั้งสองข้าง

คำถาม: จงแก้อสมการ $5x - 2 \geq 3x + 8$ และแสดงคำตอบในรูปสัญกรณ์ช่วง

แนวคิด:

  • ย้ายพจน์ตัวแปรไปอยู่ข้างหนึ่ง และค่าคงที่ไปอีกข้างหนึ่ง
  • แก้สমการที่เหลือ

เฉลย:

$5x - 2 \geq 3x + 8$

ลบ $3x$ ทั้งสองข้าง:

$5x - 3x - 2 \geq 3x - 3x + 8$

$2x - 2 \geq 8$

บวก 2 ทั้งสองข้าง:

$2x - 2 + 2 \geq 8 + 2$

$2x \geq 10$

หารด้วย 2:

$\frac{2x}{2} \geq \frac{10}{2}$

$x \geq 5$

สัญกรณ์ช่วง: $[5, \infty)$

โจทย์ข้อที่ 4: การแก้อสมการที่มีวงเล็บ

คำถาม: จงแก้อสมการ $3(x + 1) - 2x < 5x - 1$

แนวคิด:

  • กระจายค่าที่อยู่นอกวงเล็บเข้าไป
  • รวมพจน์ที่คล้ายกัน
  • ย้ายพจน์ตัวแปรและค่าคงที่ไปคนละข้าง
  • ระวังการคูณ/หารด้วยจำนวนลบ

เฉลย:

$3(x + 1) - 2x < 5x - 1$

กระจาย 3 เข้าไป:

$3x + 3 - 2x < 5x - 1$

รวมพจน์ $x$ ทางซ้าย:

$x + 3 < 5x - 1$

ลบ $x$ ทั้งสองข้าง:

$x - x + 3 < 5x - x - 1$

$3 < 4x - 1$

บวก 1 ทั้งสองข้าง:

$3 + 1 < 4x - 1 + 1$

$4 < 4x$

หารด้วย 4:

$\frac{4}{4} < \frac{4x}{4}$

$1 < x$ หรือ $x > 1$

สัญกรณ์ช่วง: $(1, \infty)$

โจทย์ข้อที่ 5: การแก้อสมการแบบสองชั้น

คำถาม: จงแก้อสมการ $-5 \leq 2x + 1 < 9$

แนวคิด:

  • แก้อสมการแยกกัน หรือ
  • ดำเนินการทุกส่วนของอสมการพร้อมกัน (บวก/ลบ/คูณ/หาร)

เฉลย:

$-5 \leq 2x + 1 < 9$

ลบ 1 ตลอดอสมการ:

$-5 - 1 \leq 2x + 1 - 1 < 9 - 1$

$-6 \leq 2x < 8$

หารด้วย 2 ตลอดอสมการ:

$\frac{-6}{2} \leq \frac{2x}{2} < \frac{8}{2}$

$-3 \leq x < 4$

สัญกรณ์ช่วง: $[-3, 4)$

4.1.5 ตัวอย่างข้อสอบฝึกทำ

ข้อที่ 1

จงแก้อสมการ $3x - 4 > 11$ และแสดงคำตอบในรูปสัญกรณ์ช่วง

แนวคิด

ย้ายพจน์ที่เป็นค่าคงที่ไปอีกฝั่งด้วยการบวก/ลบ

กำจัดสัมประสิทธิ์หน้าตัวแปรด้วยการคูณ/หาร (ระวังเครื่องหมาย)

เฉลย

$3x - 4 > 11$

บวก 4 ทั้งสองข้าง:

$3x > 11 + 4$

$3x > 15$

หารด้วย 3 (จำนวนบวก):

$x > \frac{15}{3}$

$x > 5$

คำตอบ: $(5, \infty)$

ข้อที่ 2

จงแก้อสมการ $-2x + 5 \leq 15$ และแสดงคำตอบในรูปสัญกรณ์ช่วง

แนวคิด

ย้ายพจน์ที่เป็นค่าคงที่ไปอีกฝั่ง

กำจัดสัมประสิทธิ์หน้าตัวแปร (ต้องกลับเครื่องหมายอสมการเมื่อคูณหรือหารด้วยจำนวนลบ)

เฉลย

$-2x + 5 \leq 15$

ลบ 5 ทั้งสองข้าง:

$-2x \leq 15 - 5$

$-2x \leq 10$

หารด้วย -2 (จำนวนลบ - กลับเครื่องหมาย):

$x \geq \frac{10}{-2}$

$x \geq -5$

คำตอบ: $[-5, \infty)$

ข้อที่ 3

จงแก้อสมการ $7x + 2 < 4x - 7$ และแสดงคำตอบในรูปสัญกรณ์ช่วง

แนวคิด

ย้ายพจน์ตัวแปรไปอยู่ข้างหนึ่ง และค่าคงที่ไปอีกข้างหนึ่ง

แก้สมการที่เหลือ

เฉลย

$7x + 2 < 4x - 7$

ลบ $4x$ ทั้งสองข้าง:

$7x - 4x + 2 < -7$

$3x + 2 < -7$

ลบ 2 ทั้งสองข้าง:

$3x < -7 - 2$

$3x < -9$

หารด้วย 3:

$x < \frac{-9}{3}$

$x < -3$

คำตอบ: $(-\infty, -3)$

ข้อที่ 4

จงแก้อสมการ $4(x - 3) \geq 2x + 6$ และแสดงคำตอบในรูปสัญกรณ์ช่วง

แนวคิด

กระจายค่าที่อยู่นอกวงเล็บเข้าไป

รวมพจน์ที่คล้ายกัน

ย้ายพจน์ตัวแปรและค่าคงที่ไปคนละข้าง

ระวังการคูณ/หารด้วยจำนวนลบ

เฉลย

$4(x - 3) \geq 2x + 6$

กระจาย 4 เข้าไป:

$4x - 12 \geq 2x + 6$

ลบ $2x$ ทั้งสองข้าง:

$4x - 2x - 12 \geq 6$

$2x - 12 \geq 6$

บวก 12 ทั้งสองข้าง:

$2x \geq 6 + 12$

$2x \geq 18$

หารด้วย 2:

$x \geq \frac{18}{2}$

$x \geq 9$

คำตอบ: $[9, \infty)$

ข้อที่ 5

จงแก้อสมการ $-10 < 3x + 2 \leq 8$ และแสดงคำตอบในรูปสัญกรณ์ช่วง

แนวคิด

ดำเนินการทุกส่วนของอสมการพร้อมกัน (บวก/ลบ/คูณ/หาร)

เฉลย

$-10 < 3x + 2 \leq 8$

ลบ 2 ตลอดอสมการ:

$-10 - 2 < 3x + 2 - 2 \leq 8 - 2$

$-12 < 3x \leq 6$

หารด้วย 3 ตลอดอสมการ:

$\frac{-12}{3} < \frac{3x}{3} \leq \frac{6}{3}$

$-4 < x \leq 2$

คำตอบ: $(-4, 2]$

ข้อที่ 6

จงแก้อสมการ $\frac{x}{3} - \frac{1}{2} > \frac{5}{6}$ และแสดงคำตอบในรูปสัญกรณ์ช่วง

แนวคิด

หา ค.ร.น. ของตัวส่วนทั้งหมด

คูณอสมการด้วย ค.ร.น. เพื่อกำจัดตัวส่วน

แก้อสมการเชิงเส้นที่เหลือ

เฉลย

$\frac{x}{3} - \frac{1}{2} > \frac{5}{6}$

ค.ร.น. ของ 3, 2, 6 คือ 6

คูณอสมการด้วย 6:

$6\left(\frac{x}{3} - \frac{1}{2}\right) > 6\left(\frac{5}{6}\right)$

$2x - 3 > 5$

บวก 3 ทั้งสองข้าง:

$2x > 5 + 3$

$2x > 8$

หารด้วย 2:

$x > \frac{8}{2}$

$x > 4$

คำตอบ: $(4, \infty)$

ข้อที่ 7

จงแก้อสมการต่อไปนี้:

ก. $2(x + 1) > 2x + 5$

ข. $3x - 7 \leq 3x + 2$

แนวคิด

แก้อสมการตามปกติ

หากตัวแปรหายไป ให้พิจารณาว่าอสมการที่เหลือเป็นจริงหรือเท็จ

เฉลย

ก. $2(x + 1) > 2x + 5$

$2x + 2 > 2x + 5$

ลบ $2x$ ทั้งสองข้าง:

$2 > 5$

ข้อความ "$2 > 5$" เป็นเท็จเสมอ ไม่ว่า $x$ จะเป็นค่าใด

ดังนั้น อสมการนี้ ไม่มีคำตอบ (เซตว่าง: $\emptyset$)

ข. $3x - 7 \leq 3x + 2$

ลบ $3x$ ทั้งสองข้าง:

$-7 \leq 2$

ข้อความ "$-7 \leq 2$" เป็นจริงเสมอ ไม่ว่า $x$ จะเป็นค่าใด

ดังนั้น อสมการนี้เป็นจริงสำหรับ ทุกจำนวนจริง

คำตอบ: ก. $\emptyset$ (เซตว่าง), ข. $(-\infty, \infty)$ (จำนวนจริงทั้งหมด)

ข้อที่ 8

จงแก้อสมการ $5x - 1 \neq 9$ และแสดงคำตอบในรูปสัญกรณ์ช่วง

แนวคิด

แก้อสมการเหมือนสมการปกติ แต่แทนที่จะเป็น = ให้เป็น $\neq$

คำตอบคือจำนวนจริงทั้งหมด ยกเว้นค่าที่ทำให้สมการเท่ากับ

เฉลย

$5x - 1 \neq 9$

บวก 1 ทั้งสองข้าง:

$5x \neq 9 + 1$

$5x \neq 10$

หารด้วย 5:

$x \neq \frac{10}{5}$

$x \neq 2$

คำตอบคือจำนวนจริงทั้งหมด ยกเว้น 2

คำตอบ: $(-\infty, 2) \cup (2, \infty)$

ข้อที่ 9

จงแก้อสมการ $\frac{x + 1}{2} - \frac{x - 2}{3} \geq 1$ และแสดงคำตอบในรูปสัญกรณ์ช่วง

แนวคิด

หา ค.ร.น. ของตัวส่วนทั้งหมด

คูณอสมการด้วย ค.ร.น. เพื่อกำจัดตัวส่วน (ระวังวงเล็บเมื่อมีหลายพจน์ในเศษ)

แก้อสมการเชิงเส้นที่เหลือ

เฉลย

$\frac{x + 1}{2} - \frac{x - 2}{3} \geq 1$

ค.ร.น. ของ 2, 3 คือ 6

คูณอสมการด้วย 6:

$6\left(\frac{x + 1}{2}\right) - 6\left(\frac{x - 2}{3}\right) \geq 6(1)$

$3(x + 1) - 2(x - 2) \geq 6$

กระจายค่าเข้าวงเล็บ:

$3x + 3 - (2x - 4) \geq 6$

$3x + 3 - 2x + 4 \geq 6$

รวมพจน์ที่คล้ายกัน:

$x + 7 \geq 6$

ลบ 7 ทั้งสองข้าง:

$x \geq 6 - 7$

$x \geq -1$

คำตอบ: $[-1, \infty)$

ข้อที่ 10

ร้านขนมปังแห่งหนึ่งขายขนมปังเนยสดชิ้นละ 25 บาท และมีต้นทุนในการผลิตขนมปังชิ้นละ 10 บาท โดยมีค่าใช้จ่ายคงที่ของร้านอยู่ที่ 1,500 บาทต่อวัน หากร้านต้องการให้มีกำไรอย่างน้อย 500 บาทต่อวัน จะต้องขายขนมปังอย่างน้อยกี่ชิ้น?

แนวคิด

กำหนดตัวแปรแทนจำนวนขนมปังที่ขาย

สร้างฟังก์ชันกำไร: กำไร = รายรับรวม - ต้นทุนรวม

รายรับรวม = (ราคาขายต่อชิ้น) × (จำนวนชิ้น)

ต้นทุนรวม = (ต้นทุนผลิตต่อชิ้น) × (จำนวนชิ้น) + ค่าใช้จ่ายคงที่

ตั้งอสมการตามเงื่อนไขกำไรที่ต้องการ

แก้อสมการเพื่อหาจำนวนชิ้นขนมปัง

เฉลย

ให้ $x$ แทนจำนวนขนมปังที่ขาย (ชิ้น)

รายรับรวม: $25x$ บาท

ต้นทุนแปรผัน: $10x$ บาท

ต้นทุนคงที่: $1,500$ บาท

กำไร: รายรับรวม - (ต้นทุนแปรผัน + ต้นทุนคงที่)

กำไร $= 25x - (10x + 1500)$

กำไร $= 25x - 10x - 1500$

กำไร $= 15x - 1500$

ร้านต้องการกำไรอย่างน้อย 500 บาท:

$15x - 1500 \geq 500$

บวก 1500 ทั้งสองข้าง:

$15x \geq 500 + 1500$

$15x \geq 2000$

หารด้วย 15:

$x \geq \frac{2000}{15}$

$x \geq \frac{400}{3}$

$x \geq 133.33...$

เนื่องจากจำนวนขนมปังต้องเป็นจำนวนเต็มและต้องขาย "อย่างน้อย" ที่สุดตามที่กำหนด

จะต้องขายอย่างน้อย 134 ชิ้น

คำตอบ: 134 ชิ้น
