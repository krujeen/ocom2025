3.1.1 ความสัมพันธ์ (Relations)

นิยาม: ความสัมพันธ์

ความสัมพันธ์ คือ เซตของคู่อันดับ (ordered pairs) โดยที่สมาชิกตัวแรกของคู่อันดับมาจากเซตหนึ่ง และสมาชิกตัวที่สองมาจากอีกเซตหนึ่ง

ผลคูณคาร์ทีเซียน (Cartesian Product)

ถ้า $A$ และ $B$ เป็นเซตใดๆ ผลคูณคาร์ทีเซียน (Cartesian Product) ของ $A$ และ $B$ เขียนแทนด้วย $A \times B$ คือเซตของคู่อันดับ $(a,b)$ ทั้งหมด โดยที่ $a \in A$ และ $b \in B$

$$A \times B = \{(a,b) \mid a \in A \text{ and } b \in B\}$$

ความสัมพันธ์ $r$ จากเซต $A$ ไปยังเซต $B$ คือสับเซตของ $A \times B$

โดเมนและเรนจ์ของความสัมพันธ์

  • โดเมน (Domain) ของความสัมพันธ์ $r$: เซตของสมาชิกตัวแรกของทุกคู่อันดับใน $r$ เขียนแทนด้วย $D_r$
    $$D_r = \{a \mid (a,b) \in r \text{ for some } b\}$$
  • เรนจ์ (Range) ของความสัมพันธ์ $r$: เซตของสมาชิกตัวที่สองของทุกคู่อันดับใน $r$ เขียนแทนด้วย $R_r$
    $$R_r = \{b \mid (a,b) \in r \text{ for some } a\}$$

ตัวอย่าง ความสัมพันธ์

ให้ $A = \{1,2\}$, $B = \{a,b\}$

$A \times B = \{(1,a), (1,b), (2,a), (2,b)\}$

ความสัมพันธ์ $r = \{(1,a), (2,b)\}$

$D_r = \{1,2\}$

$R_r = \{a,b\}$

3.1.2 ฟังก์ชัน (Functions)

นิยาม: ฟังก์ชัน

ฟังก์ชัน คือ ความสัมพันธ์ชนิดพิเศษที่สมาชิกตัวแรกแต่ละตัว (จากโดเมน) จับคู่กับสมาชิกตัวที่สองเพียงตัวเดียวเท่านั้น (ในเรนจ์) หรือกล่าวอีกนัยหนึ่งคือ ไม่มีคู่อันดับสองคู่ใดในฟังก์ชันที่มีสมาชิกตัวแรกซ้ำกันแต่มีสมาชิกตัวที่สองต่างกัน

การทดสอบเส้นแนวดิ่ง (Vertical Line Test)

ใช้ตรวจสอบว่ากราฟที่กำหนดให้เป็นฟังก์ชันหรือไม่ โดยการลากเส้นแนวดิ่ง (เส้นตรงขนานกับแกน Y) ตัดกราฟ

  • ถ้าเส้นแนวดิ่งตัดกราฟเพียงจุดเดียวสำหรับทุกเส้นแนวดิ่ง ➜ กราฟนั้นเป็นฟังก์ชัน
  • ถ้ามีเส้นแนวดิ่งเส้นใดที่ตัดกราฟมากกว่าหนึ่งจุด ➜ กราฟนั้นไม่ใช่ฟังก์ชัน

ตัวอย่าง ฟังก์ชันและไม่ใช่ฟังก์ชัน

ตัวอย่างฟังก์ชัน:

  • $f = \{(1,2), (2,4), (3,6)\}$ (เป็นฟังก์ชัน)
  • $f(x) = x^2$ (เป็นฟังก์ชัน)
  • $y = x^3 - 1$ (เป็นฟังก์ชัน)

ตัวอย่างที่ไม่ใช่ฟังก์ชัน:

  • $r = \{(1,2), (1,3), (2,4)\}$ (ไม่เป็นฟังก์ชัน เพราะ 1 จับคู่กับ 2 และ 3)
  • $x^2 + y^2 = 25$ (ไม่เป็นฟังก์ชัน เพราะ $x = 0$ จะได้ $y = ±5$)

3.1.3 ฟังก์ชันกำลังสอง (Quadratic Functions) และกราฟ

นิยาม: ฟังก์ชันกำลังสอง

ฟังก์ชันกำลังสอง คือ ฟังก์ชันที่เขียนอยู่ในรูปทั่วไป

$$y = f(x) = ax^2 + bx + c$$

เมื่อ $a, b, c$ เป็นค่าคงที่ใดๆ และ $a \neq 0$

กราฟของฟังก์ชันกำลังสอง คือ รูปพาราโบลา

ลักษณะสำคัญของกราฟพาราโบลา

1. การเปิดของกราฟ (หงาย/คว่ำ)

  • ถ้า $a > 0$ ➜ กราฟพาราโบลาจะ หงายขึ้น (มีจุดต่ำสุด)
  • ถ้า $a < 0$ ➜ กราฟพาราโบลาจะ คว่ำลง (มีจุดสูงสุด)

2. จุดยอด (Vertex)

คือจุดสูงสุดหรือจุดต่ำสุดของพาราโบลา

$$\text{พิกัดของจุดยอดคือ } (h,k) = \left(-\frac{b}{2a}, f\left(-\frac{b}{2a}\right)\right)$$ $$\text{หรือ } \left(-\frac{b}{2a}, \frac{4ac - b^2}{4a}\right)$$
  • จุดนี้เป็นจุดที่ทำให้เกิด ค่าต่ำสุด ของฟังก์ชัน (เมื่อ $a > 0$)
  • หรือ ค่าสูงสุด ของฟังก์ชัน (เมื่อ $a < 0$)

3. แกนสมมาตร (Axis of Symmetry)

คือเส้นตรงแนวดิ่งที่ลากผ่านจุดยอดและแบ่งพาราโบลาออกเป็นสองส่วนที่สมมาตรกัน

$$\text{สมการของแกนสมมาตรคือ } x = -\frac{b}{2a}$$

4. จุดตัดแกน Y (Y-intercept)

  • เกิดขึ้นเมื่อ $x = 0$
  • แทนค่า $x = 0$ ลงในสมการจะได้ $y = a(0)^2 + b(0) + c = c$
  • พิกัดจุดตัดแกน Y คือ $(0, c)$

5. จุดตัดแกน X (X-intercepts)

  • เกิดขึ้นเมื่อ $y = 0$
  • แทนค่า $y = 0$ ลงในสมการ: $ax^2 + bx + c = 0$
  • สามารถหาค่า $x$ ได้โดยการแยกตัวประกอบ หรือใช้สูตร $x = \frac{-b ± \sqrt{b^2 - 4ac}}{2a}$

จำนวนจุดตัดแกน X ขึ้นอยู่กับค่า $D = b^2 - 4ac$ (discriminant):

  • ถ้า $D > 0$: มี 2 จุดตัดแกน X (รากที่แตกต่างกัน 2 ราก)
  • ถ้า $D = 0$: มี 1 จุดตัดแกน X (กราฟสัมผัสแกน X ที่จุดยอด, รากซ้ำ 1 ราก)
  • ถ้า $D < 0$: ไม่มีจุดตัดแกน X (รากเป็นจำนวนเชิงซ้อน)

3.1.4 ตัวอย่างโจทย์และแนวทางการทำข้อสอบ

โจทย์ข้อที่ 1: การพิจารณาความสัมพันธ์และฟังก์ชัน

คำถาม: กำหนดให้เซต $A = \{1,2,3,4\}$ และเซต $B = \{a,b,c\}$ จงพิจารณาว่าความสัมพันธ์ใดต่อไปนี้ เป็นฟังก์ชัน จาก $A$ ไป $B$

ก. $r_1 = \{(1,a), (2,b), (3,c), (4,c)\}$

ข. $r_2 = \{(1,a), (1,b), (2,c), (3,a), (4,b)\}$

ค. $r_3 = \{(1,a), (2,b), (3,c)\}$

แนวคิด: ฟังก์ชันคือความสัมพันธ์ที่สมาชิกตัวแรกแต่ละตัว (จากโดเมน) จับคู่กับสมาชิกตัวที่สองเพียงตัวเดียว

เฉลย:

ก. $r_1 = \{(1,a), (2,b), (3,c), (4,c)\}$: เป็นฟังก์ชัน เพราะสมาชิกตัวแรกแต่ละตัว (1, 2, 3, 4) จับคู่กับสมาชิกตัวที่สองเพียงตัวเดียว

ข. $r_2 = \{(1,a), (1,b), (2,c), (3,a), (4,b)\}$: ไม่เป็นฟังก์ชัน เพราะสมาชิกตัวแรก 1 จับคู่กับทั้ง $a$ และ $b$

ค. $r_3 = \{(1,a), (2,b), (3,c)\}$: เป็นฟังก์ชัน แต่ไม่เป็นฟังก์ชันจาก $A$ ทั่วถึง $B$ (เนื่องจากสมาชิก 4 ใน $A$ ไม่ได้ถูกจับคู่)

คำตอบ: ก. และ ค.

โจทย์ข้อที่ 2: การหาโดเมนและเรนจ์ของความสัมพันธ์

คำถาม: จงหาโดเมนและเรนจ์ของความสัมพันธ์ $r = \{(x,y) \mid y = \sqrt{x-2}\}$

แนวคิด:

  • โดเมน $(D_r)$: พิจารณาค่า $x$ ที่ทำให้ $y$ เป็นจำนวนจริง (เงื่อนไขของ $\sqrt{\;}$ คือสิ่งที่อยู่ข้างในต้อง $≥ 0$)
  • เรนจ์ $(R_r)$: พิจารณาค่า $y$ ที่เป็นไปได้จากเงื่อนไขของ $\sqrt{\;}$ (ค่าที่ได้จาก $\sqrt{\;}$ ต้อง $≥ 0$)

เฉลย:

โดเมน $(D_r)$:

เนื่องจาก $y = \sqrt{x-2}$ และ $y$ ต้องเป็นจำนวนจริง ดังนั้นสิ่งที่อยู่ใต้เครื่องหมายกรณฑ์ต้องไม่ติดลบ:

$x - 2 ≥ 0$

$x ≥ 2$

ดังนั้น $D_r = \{x \mid x ≥ 2\} = [2, ∞)$

เรนจ์ $(R_r)$:

จาก $y = \sqrt{x-2}$ และเรารู้ว่า $x - 2 ≥ 0$

ค่าของรากที่สองของจำนวนจริงใดๆ จะต้องเป็นจำนวนจริงที่ไม่เป็นลบเสมอ:

$\sqrt{x-2} ≥ 0$

ดังนั้น $y ≥ 0$

$R_r = \{y \mid y ≥ 0\} = [0, ∞)$

คำตอบ: $D_r = [2, ∞)$, $R_r = [0, ∞)$

โจทย์ข้อที่ 3: การวิเคราะห์กราฟฟังก์ชันกำลังสอง

คำถาม: กำหนดฟังก์ชันกำลังสอง $f(x) = -x^2 + 4x - 3$ จงหาพิกัดจุดยอดของกราฟ และระบุว่ากราฟพาราโบลาเปิดในทิศทางใด พร้อมทั้งหาค่าสูงสุดหรือต่ำสุดของฟังก์ชัน

แนวคิด:

  • ระบุค่า $a, b, c$ จากฟังก์ชัน $f(x) = ax^2 + bx + c$
  • หาพิกัดจุดยอด $(h,k) = \left(-\frac{b}{2a}, f\left(-\frac{b}{2a}\right)\right)$
  • พิจารณาเครื่องหมายของ $a$ เพื่อระบุทิศทางการเปิดและค่าสูงสุด/ต่ำสุด

เฉลย:

จาก $f(x) = -x^2 + 4x - 3$ เราได้ $a = -1$, $b = 4$, $c = -3$

ทิศทางการเปิด: เนื่องจาก $a = -1 < 0$ กราฟพาราโบลาจะ คว่ำลง และมี จุดสูงสุด

พิกัดจุดยอด:

หาพิกัด $x$ ของจุดยอด: $h = -\frac{b}{2a} = -\frac{4}{2(-1)} = -\frac{4}{-2} = 2$

หาพิกัด $y$ ของจุดยอด (แทน $x = 2$ ใน $f(x)$): $k = f(2) = -(2)^2 + 4(2) - 3 = -4 + 8 - 3 = 1$

ดังนั้น จุดยอดคือ $(2, 1)$

ค่าสูงสุด/ต่ำสุด: เนื่องจากกราฟคว่ำลง จุดยอดคือจุดสูงสุด ดังนั้น ค่าสูงสุด ของฟังก์ชันคือ 1

คำตอบ: จุดยอด $(2,1)$, กราฟคว่ำลง, ค่าสูงสุดของฟังก์ชันคือ 1

โจทย์ข้อที่ 4: การหาจุดตัดแกนของฟังก์ชันกำลังสอง

คำถาม: จงหาจุดตัดแกน X และจุดตัดแกน Y ของกราฟฟังก์ชัน $y = x^2 - 5x + 6$

แนวคิด:

  • จุดตัดแกน Y: ให้ $x = 0$ แล้วหาค่า $y$
  • จุดตัดแกน X: ให้ $y = 0$ แล้วแก้สมการกำลังสอง $x^2 - 5x + 6 = 0$ (โดยการแยกตัวประกอบหรือใช้สูตร)

เฉลย:

จุดตัดแกน Y:

ให้ $x = 0$:

$y = (0)^2 - 5(0) + 6 = 6$

จุดตัดแกน Y คือ $(0, 6)$

จุดตัดแกน X:

ให้ $y = 0$:

$x^2 - 5x + 6 = 0$

แยกตัวประกอบ:

$(x - 2)(x - 3) = 0$

ดังนั้น $x - 2 = 0 ⟹ x = 2$

หรือ $x - 3 = 0 ⟹ x = 3$

จุดตัดแกน X คือ $(2, 0)$ และ $(3, 0)$

คำตอบ: จุดตัดแกน Y คือ $(0, 6)$, จุดตัดแกน X คือ $(2, 0)$ และ $(3, 0)$

โจทย์ข้อที่ 5: การสร้างสมการพาราโบลาจากข้อมูลที่กำหนด

คำถาม: ฟังก์ชันกำลังสองรูปหนึ่งมีจุดยอดอยู่ที่ $(-1, -4)$ และผ่านจุด $(1, 0)$ จงหาสมการของฟังก์ชันนี้

แนวคิด:

  • ใช้รูปแบบสมการพาราโบลาที่รู้จุดยอด: $y = a(x - h)^2 + k$
  • แทนพิกัดจุดยอด $(h, k)$ ลงในสมการ
  • แทนพิกัดจุดที่กราฟผ่าน $(x, y)$ ลงในสมการเพื่อหาค่า $a$
  • เขียนสมการสุดท้าย

เฉลย:

1. ใช้รูปแบบสมการพาราโบลาที่รู้จุดยอด: $y = a(x - h)^2 + k$

จุดยอดคือ $(-1, -4)$ ดังนั้น $h = -1$, $k = -4$

แทนค่าลงในสมการ: $y = a(x - (-1))^2 + (-4)$

$y = a(x + 1)^2 - 4$

2. กราฟผ่านจุด $(1, 0)$ ดังนั้นแทน $x = 1$, $y = 0$ ลงในสมการเพื่อหา $a$:

$0 = a(1 + 1)^2 - 4$

$0 = a(2)^2 - 4$

$0 = 4a - 4$

$4a = 4$

$a = 1$

3. แทนค่า $a = 1$ กลับเข้าไปในสมการ:

$y = 1(x + 1)^2 - 4$

$y = (x^2 + 2x + 1) - 4$

$y = x^2 + 2x - 3$

คำตอบ: $y = x^2 + 2x - 3$

3.1.5 ตัวอย่างข้อสอบฝึกทำ

ข้อที่ 1

กำหนดให้เซต $A = \{a,b,c\}$ และเซต $B = \{1,2,3,4\}$ จงพิจารณาว่าความสัมพันธ์ใดต่อไปนี้ เป็นฟังก์ชัน จากเซตหนึ่งไปยังอีกเซตหนึ่ง?

ก. $r_1 = \{(a,1), (b,2), (c,3)\}$

ข. $r_2 = \{(a,1), (a,2), (b,3), (c,4)\}$

ค. $r_3 = \{(1,a), (2,b), (3,c), (4,b)\}$

ง. $r_4 = \{(a,1), (b,1), (c,1)\}$

แนวคิด

ฟังก์ชันคือความสัมพันธ์ที่สมาชิกตัวแรกแต่ละตัว (จากโดเมน) จับคู่กับสมาชิกตัวที่สองเพียงตัวเดียวเท่านั้น

เฉลย

ก. $r_1 = \{(a,1), (b,2), (c,3)\}$: เป็นฟังก์ชัน เพราะสมาชิกตัวแรกแต่ละตัว $(a,b,c)$ จับคู่กับสมาชิกตัวที่สองเพียงตัวเดียว

ข. $r_2 = \{(a,1), (a,2), (b,3), (c,4)\}$: ไม่เป็นฟังก์ชัน เพราะสมาชิกตัวแรก $a$ จับคู่กับสมาชิกตัวที่สองถึงสองตัวคือ 1 และ 2

ค. $r_3 = \{(1,a), (2,b), (3,c), (4,b)\}$: เป็นฟังก์ชัน เพราะสมาชิกตัวแรกแต่ละตัว $(1,2,3,4)$ จับคู่กับสมาชิกตัวที่สองเพียงตัวเดียว

ง. $r_4 = \{(a,1), (b,1), (c,1)\}$: เป็นฟังก์ชัน เพราะสมาชิกตัวแรกแต่ละตัว $(a,b,c)$ จับคู่กับสมาชิกตัวที่สองเพียงตัวเดียว

คำตอบ: ก., ค., ง.

ข้อที่ 2

จงหาโดเมนและเรนจ์ของความสัมพันธ์ $r = \{(x,y) \mid y = \frac{1}{x-4}\}$

แนวคิด

โดเมน $(D_r)$: พิจารณาค่า $x$ ที่ทำให้พจน์ในส่วนไม่เป็นศูนย์

เรนจ์ $(R_r)$: พิจารณาค่า $y$ ที่เป็นไปได้เมื่อ $x$ เป็นสมาชิกในโดเมน

เฉลย

โดเมน $(D_r)$:

เนื่องจาก $y = \frac{1}{x-4}$ และส่วนของเศษส่วนไม่สามารถเป็นศูนย์ได้:

$x - 4 \neq 0$

$x \neq 4$

ดังนั้น $D_r = \{x \mid x \in \mathbb{R}, x \neq 4\}$ หรือ $(-∞,4) ∪ (4,∞)$

เรนจ์ $(R_r)$:

จาก $y = \frac{1}{x-4}$

เราสามารถเขียน $x - 4 = \frac{1}{y}$

ดังนั้น $x = 4 + \frac{1}{y}$

เนื่องจาก $x$ เป็นจำนวนจริงใดๆ (ยกเว้น 4) และ $y$ อยู่ในส่วนของเศษส่วน $y \neq 0$

ดังนั้น $R_r = \{y \mid y \in \mathbb{R}, y \neq 0\}$ หรือ $(-∞,0) ∪ (0,∞)$

คำตอบ: $D_r = \{x \mid x \in \mathbb{R}, x \neq 4\}$, $R_r = \{y \mid y \in \mathbb{R}, y \neq 0\}$

ข้อที่ 3

กำหนดฟังก์ชันกำลังสอง $f(x) = 2x^2 - 8x + 6$ จงหาพิกัดจุดยอดของกราฟ, ระบุว่ากราฟพาราโบลาเปิดในทิศทางใด พร้อมทั้งหาค่าสูงสุดหรือต่ำสุดของฟังก์ชัน

แนวคิด

ระบุค่า $a, b, c$ จากฟังก์ชัน $f(x) = ax^2 + bx + c$

หาพิกัดจุดยอด $(h,k) = \left(-\frac{b}{2a}, f\left(-\frac{b}{2a}\right)\right)$

พิจารณาเครื่องหมายของ $a$ เพื่อระบุทิศทางการเปิดและค่าสูงสุด/ต่ำสุด

เฉลย

จาก $f(x) = 2x^2 - 8x + 6$ เราได้ $a = 2$, $b = -8$, $c = 6$

ทิศทางการเปิด: เนื่องจาก $a = 2 > 0$ กราฟพาราโบลาจะ หงายขึ้น และมี จุดต่ำสุด

พิกัดจุดยอด:

หาพิกัด $x$ ของจุดยอด: $h = -\frac{b}{2a} = -\frac{-8}{2(2)} = \frac{8}{4} = 2$

หาพิกัด $y$ ของจุดยอด (แทน $x = 2$ ใน $f(x)$): $k = f(2) = 2(2)^2 - 8(2) + 6 = 2(4) - 16 + 6 = 8 - 16 + 6 = -2$

ดังนั้น จุดยอดคือ $(2, -2)$

ค่าสูงสุด/ต่ำสุด: เนื่องจากกราฟหงายขึ้น จุดยอดคือจุดต่ำสุด ดังนั้น ค่าต่ำสุด ของฟังก์ชันคือ $-2$

คำตอบ: จุดยอด $(2, -2)$, กราฟหงายขึ้น, ค่าต่ำสุดของฟังก์ชันคือ $-2$

ข้อที่ 4

จงหาจุดตัดแกน X และจุดตัดแกน Y ของกราฟฟังก์ชัน $y = x^2 - x - 12$

แนวคิด

จุดตัดแกน Y: ให้ $x = 0$ แล้วหาค่า $y$

จุดตัดแกน X: ให้ $y = 0$ แล้วแก้สมการกำลังสอง $x^2 - x - 12 = 0$ (โดยการแยกตัวประกอบหรือใช้สูตร)

เฉลย

จุดตัดแกน Y:

ให้ $x = 0$:

$y = (0)^2 - (0) - 12 = -12$

จุดตัดแกน Y คือ $(0, -12)$

จุดตัดแกน X:

ให้ $y = 0$:

$x^2 - x - 12 = 0$

แยกตัวประกอบ:

$(x - 4)(x + 3) = 0$

ดังนั้น $x - 4 = 0 ⟹ x = 4$

หรือ $x + 3 = 0 ⟹ x = -3$

จุดตัดแกน X คือ $(4, 0)$ และ $(-3, 0)$

คำตอบ: จุดตัดแกน Y คือ $(0, -12)$, จุดตัดแกน X คือ $(4, 0)$ และ $(-3, 0)$

ข้อที่ 5

ฟังก์ชันกำลังสองรูปหนึ่งมีแกนสมมาตรคือเส้นตรง $x = 3$ และมีจุดต่ำสุดของกราฟที่ $y = -5$ ถ้ากราฟนี้ผ่านจุด $(1, 3)$ จงหาสมการของฟังก์ชันนี้

แนวคิด

จากแกนสมมาตร $(x = h)$ และจุดต่ำสุด/สูงสุด $(y = k)$ จะได้พิกัดจุดยอด $(h, k)$

ใช้รูปแบบสมการพาราโบลาที่รู้จุดยอด: $y = a(x - h)^2 + k$

แทนพิกัดจุดที่กราฟผ่าน $(x, y)$ ลงในสมการเพื่อหาค่า $a$

เฉลย

จากแกนสมมาตร $x = 3$ และจุดต่ำสุด $y = -5$ แสดงว่าจุดยอดคือ $(3, -5)$ ดังนั้น $h = 3$, $k = -5$

แทนค่าลงในสมการ $y = a(x - h)^2 + k$:

$y = a(x - 3)^2 - 5$

กราฟผ่านจุด $(1, 3)$ ดังนั้นแทน $x = 1$, $y = 3$ ลงในสมการเพื่อหา $a$:

$3 = a(1 - 3)^2 - 5$

$3 = a(-2)^2 - 5$

$3 = 4a - 5$

$8 = 4a$

$a = 2$

แทนค่า $a = 2$ กลับเข้าไปในสมการ:

$y = 2(x - 3)^2 - 5$

$y = 2(x^2 - 6x + 9) - 5$

$y = 2x^2 - 12x + 18 - 5$

$y = 2x^2 - 12x + 13$

คำตอบ: $y = 2x^2 - 12x + 13$

ข้อที่ 6

สำหรับฟังก์ชัน $f(x) = x^2 + 6x + c$ จงหาค่าของ $c$ ที่ทำให้กราฟของฟังก์ชัน สัมผัสแกน X เพียงจุดเดียว

แนวคิด

กราฟฟังก์ชันกำลังสองสัมผัสแกน X เพียงจุดเดียวเมื่อสมการ $ax^2 + bx + c = 0$ มีรากซ้ำกันหนึ่งราก

เงื่อนไขนี้เกิดขึ้นเมื่อค่า Discriminant $D = b^2 - 4ac = 0$

เฉลย

จากฟังก์ชัน $f(x) = x^2 + 6x + c$ เราได้ $a = 1$, $b = 6$, $c = c$

กราฟสัมผัสแกน X เพียงจุดเดียวเมื่อ $b^2 - 4ac = 0$

$(6)^2 - 4(1)(c) = 0$

$36 - 4c = 0$

$36 = 4c$

$c = \frac{36}{4} = 9$

คำตอบ: $c = 9$

ข้อที่ 7

จงหาโดเมนและเรนจ์ของฟังก์ชัน $f(x) = \sqrt{9 - x^2}$

แนวคิด

โดเมน: สิ่งที่อยู่ใต้เครื่องหมายกรณฑ์ต้อง $≥ 0$

เรนจ์: ค่าของรากที่สองที่เป็นบวกเสมอ และพิจารณาค่าสูงสุดที่ $x = 0$

เฉลย

โดเมน $(D_f)$:

สิ่งที่อยู่ใต้เครื่องหมายกรณฑ์ต้องไม่เป็นลบ:

$9 - x^2 ≥ 0$

$x^2 ≤ 9$

$-3 ≤ x ≤ 3$

ดังนั้น $D_f = [-3, 3]$

เรนจ์ $(R_f)$:

จาก $y = \sqrt{9 - x^2}$

เรารู้ว่า $y ≥ 0$ เสมอ

ค่าสูงสุดของ $y$ เกิดขึ้นเมื่อ $9 - x^2$ มีค่ามากที่สุด ซึ่งคือเมื่อ $x = 0$

$y_{max} = \sqrt{9 - 0^2} = \sqrt{9} = 3$

ค่าต่ำสุดของ $y$ เกิดขึ้นเมื่อ $9 - x^2 = 0$ (คือเมื่อ $x = ±3$)

$y_{min} = \sqrt{9 - (±3)^2} = \sqrt{9 - 9} = 0$

ดังนั้น $R_f = [0, 3]$

คำตอบ: $D_f = [-3, 3]$, $R_f = [0, 3]$

ข้อที่ 8

พาราโบลา $y = ax^2 + bx + c$ ผ่านจุด $(0, 1)$, $(1, 0)$ และ $(2, 1)$ จงหาสมการของพาราโบลานี้

แนวคิด

แทนพิกัดแต่ละจุดลงในสมการ $y = ax^2 + bx + c$ เพื่อสร้างระบบสมการ

แก้ระบบสมการเพื่อหาค่า $a, b, c$

เขียนสมการสุดท้าย

เฉลย

ผ่านจุด $(0, 1)$: แทน $x = 0$, $y = 1$

$1 = a(0)^2 + b(0) + c$

$1 = c$

ผ่านจุด $(1, 0)$: แทน $x = 1$, $y = 0$ และ $c = 1$

$0 = a(1)^2 + b(1) + 1$

$0 = a + b + 1 ⟹ a + b = -1$ ... (1)

ผ่านจุด $(2, 1)$: แทน $x = 2$, $y = 1$ และ $c = 1$

$1 = a(2)^2 + b(2) + 1$

$1 = 4a + 2b + 1$

$0 = 4a + 2b ⟹ 2a + b = 0$ ... (2)

แก้ระบบสมการ (1) และ (2):

(2) - (1):

$(2a + b) - (a + b) = 0 - (-1)$

$a = 1$

แทน $a = 1$ ใน (1):

$1 + b = -1 ⟹ b = -2$

ดังนั้น $a = 1$, $b = -2$, $c = 1$

สมการพาราโบลาคือ $y = x^2 - 2x + 1$

คำตอบ: $y = x^2 - 2x + 1$

ข้อที่ 9

ฟังก์ชัน $f(x) = a(x - h)^2 + k$ มีค่าต่ำสุดเท่ากับ 5 เมื่อ $x = 3$ และ $f(1) = 9$ จงหาค่าของ $f(0)$

แนวคิด

จากค่าต่ำสุดและค่า $x$ ที่ทำให้เกิดค่าต่ำสุด จะได้พิกัดจุดยอด $(h, k)$

เมื่อรู้จุดยอดแล้ว ใช้เงื่อนไข $f(1) = 9$ เพื่อหาค่า $a$

หาค่า $f(0)$ โดยการแทน $x = 0$

เฉลย

จากเงื่อนไข "ค่าต่ำสุดเท่ากับ 5 เมื่อ $x = 3$" แสดงว่าจุดยอดคือ $(h, k) = (3, 5)$

ดังนั้น สมการของฟังก์ชันคือ $f(x) = a(x - 3)^2 + 5$

จากเงื่อนไข $f(1) = 9$:

$9 = a(1 - 3)^2 + 5$

$9 = a(-2)^2 + 5$

$9 = 4a + 5$

$4 = 4a$

$a = 1$

ดังนั้น $f(x) = (x - 3)^2 + 5$

หา $f(0)$:

$f(0) = (0 - 3)^2 + 5$

$f(0) = (-3)^2 + 5$

$f(0) = 9 + 5 = 14$

คำตอบ: $f(0) = 14$

ข้อที่ 10

นาย ก. โยนลูกบอลขึ้นไปในอากาศ ความสูงของลูกบอล (h เมตร) ณ เวลา (t วินาที) ใดๆ สามารถอธิบายได้ด้วยฟังก์ชัน $h(t) = -5t^2 + 30t + 10$ จงหาว่าลูกบอลจะขึ้นไปได้สูงสุดกี่เมตร

แนวคิด

ฟังก์ชันความสูงเป็นฟังก์ชันกำลังสอง $h(t) = at^2 + bt + c$ โดย $a = -5$, $b = 30$, $c = 10$

เนื่องจาก $a < 0$ กราฟพาราโบลาคว่ำลง ดังนั้นจุดยอดคือจุดสูงสุด

หาพิกัด $y$ ของจุดยอด $(k)$ เพื่อหาค่าความสูงสูงสุด

เฉลย

จากฟังก์ชัน $h(t) = -5t^2 + 30t + 10$ เราได้ $a = -5$, $b = 30$, $c = 10$

ลูกบอลจะขึ้นไปได้สูงสุดที่พิกัด $y$ ของจุดยอด

หาเวลาที่ลูกบอลขึ้นไปถึงจุดสูงสุด (พิกัด $t$ ของจุดยอด):

$t = -\frac{b}{2a} = -\frac{30}{2(-5)} = -\frac{30}{-10} = 3$ วินาที

หาความสูงสูงสุด (พิกัด $h$ ของจุดยอด):

แทน $t = 3$ ใน $h(t)$:

$h(3) = -5(3)^2 + 30(3) + 10$

$h(3) = -5(9) + 90 + 10$

$h(3) = -45 + 90 + 10$

$h(3) = 45 + 10 = 55$ เมตร

คำตอบ: 55 เมตร
