บทที่ 8.5: ลำดับเลขคณิตและลำดับเรขาคณิต

ทำความเข้าใจเกี่ยวกับ "ลำดับเลขคณิต" และ "ลำดับเรขาคณิต" ซึ่งเป็นลำดับของจำนวนที่มีรูปแบบการเพิ่มขึ้นหรือลดลงที่เป็นระบบ ลำดับเหล่านี้เป็นพื้นฐานสำคัญในคณิตศาสตร์และวิทยาการคอมพิวเตอร์ โดยเฉพาะอย่างยิ่งในการวิเคราะห์การเติบโต การลดลงของข้อมูล หรือการคำนวณที่เกี่ยวข้องกับการเปลี่ยนแปลงทีละขั้น

  • 8.5.2 ลำดับเรขาคณิต (Geometric Sequence)

8.5.1 ลำดับเลขคณิต (Arithmetic Sequence)

นิยาม

ลำดับเลขคณิต คือ ลำดับของจำนวนที่ผลต่างระหว่างพจน์ที่อยู่ติดกัน (พจน์ขวาตั้งลบด้วยพจน์ซ้าย) มีค่าคงที่เสมอ ค่าคงที่นี้เรียกว่า ผลต่างร่วม (common difference) เขียนแทนด้วย $d$

รูปแบบทั่วไป

$a_1, a_2, a_3, \ldots, a_n, \ldots$

โดยที่ $a_2 - a_1 = d$, $a_3 - a_2 = d$, $\ldots$, $a_n - a_{n-1} = d$

ตัวอย่าง

  • ลำดับ: $2, 5, 8, 11, \ldots$ (ผลต่างร่วม $d = 3$)
  • ลำดับ: $10, 7, 4, 1, \ldots$ (ผลต่างร่วม $d = -3$)

ก. สูตรพจน์ที่ n ของลำดับเลขคณิต

$$a_n = a_1 + (n-1)d$$

เมื่อ $a_n$ คือพจน์ที่ $n$, $a_1$ คือพจน์แรก, $n$ คือลำดับที่ของพจน์, และ $d$ คือผลต่างร่วม

ข. สูตรผลบวก n พจน์แรกของลำดับเลขคณิต

$$S_n = \frac{n}{2}(a_1 + a_n)$$

หรือ

$$S_n = \frac{n}{2}(2a_1 + (n-1)d)$$

เมื่อ $S_n$ คือผลบวกของ $n$ พจน์แรก

8.5.2 ลำดับเรขาคณิต (Geometric Sequence)

นิยาม

ลำดับเรขาคณิต คือ ลำดับของจำนวนที่อัตราส่วนระหว่างพจน์ที่อยู่ติดกัน (พจน์ขวาตั้งหารด้วยพจน์ซ้าย) มีค่าคงที่เสมอ ค่าคงที่นี้เรียกว่า อัตราส่วนร่วม (common ratio) เขียนแทนด้วย $r$

รูปแบบทั่วไป

$a_1, a_2, a_3, \ldots, a_n, \ldots$

โดยที่ $\frac{a_2}{a_1} = r$, $\frac{a_3}{a_2} = r$, $\ldots$, $\frac{a_n}{a_{n-1}} = r$

ตัวอย่าง

  • ลำดับ: $2, 6, 18, 54, \ldots$ (อัตราส่วนร่วม $r = 3$)
  • ลำดับ: $100, 50, 25, 12.5, \ldots$ (อัตราส่วนร่วม $r = 0.5$)

ก. สูตรพจน์ที่ n ของลำดับเรขาคณิต

$$a_n = a_1 r^{n-1}$$

เมื่อ $a_n$ คือพจน์ที่ $n$, $a_1$ คือพจน์แรก, $n$ คือลำดับที่ของพจน์, และ $r$ คืออัตราส่วนร่วม

ข. สูตรผลบวก n พจน์แรกของลำดับเรขาคณิต

$$S_n = \frac{a_1(r^n - 1)}{r - 1} \quad \text{เมื่อ } r \neq 1$$

หรือ

$$S_n = \frac{a_1(1 - r^n)}{1 - r} \quad \text{เมื่อ } r \neq 1$$

ถ้า $r = 1$, $S_n = n \cdot a_1$

8.5.3 ตัวอย่างโจทย์และแนวทางการทำข้อสอบ

โจทย์ข้อที่ 1: ลำดับเลขคณิต (หาพจน์ที่ n)

คำถาม: ลำดับเลขคณิตมีพจน์แรกคือ 5 และผลต่างร่วมคือ 3 จงหาพจน์ที่ 10 ของลำดับนี้

แนวคิด

  • ระบุ $a_1$, $d$, และ $n$
  • ใช้สูตร $a_n = a_1 + (n-1)d$

เฉลย

จากโจทย์: $a_1 = 5$, $d = 3$, $n = 10$

ใช้สูตรพจน์ที่ $n$: $a_n = a_1 + (n-1)d$

$a_{10} = 5 + (10-1)3$

$a_{10} = 5 + (9)3$

$a_{10} = 5 + 27$

$a_{10} = 32$

คำตอบ: 32

โจทย์ข้อที่ 2: ลำดับเลขคณิต (หาผลบวก n พจน์แรก)

คำถาม: จงหาผลบวกของ 15 พจน์แรกของลำดับเลขคณิต $2, 6, 10, 14, \ldots$

แนวคิด

  • ระบุ $a_1$, $d$, และ $n$
  • ใช้สูตร $S_n = \frac{n}{2}(2a_1 + (n-1)d)$ หรือหา $a_{15}$ ก่อนแล้วใช้ $S_n = \frac{n}{2}(a_1 + a_n)$

เฉลย

จากลำดับ: $a_1 = 2$

หาผลต่างร่วม $d = 6 - 2 = 4$

ต้องการหาผลบวก 15 พจน์แรก ดังนั้น $n = 15$

วิธีที่ 1: ใช้สูตร $S_n = \frac{n}{2}(2a_1 + (n-1)d)$

$S_{15} = \frac{15}{2}(2(2) + (15-1)4)$

$S_{15} = \frac{15}{2}(4 + (14)4)$

$S_{15} = \frac{15}{2}(4 + 56)$

$S_{15} = \frac{15}{2}(60)$

$S_{15} = 15 \times 30 = 450$

วิธีที่ 2: หา $a_{15}$ ก่อน

$a_{15} = a_1 + (15-1)d = 2 + (14)4 = 2 + 56 = 58$

$S_{15} = \frac{15}{2}(a_1 + a_{15}) = \frac{15}{2}(2 + 58) = \frac{15}{2}(60) = 15 \times 30 = 450$

คำตอบ: 450

โจทย์ข้อที่ 3: ลำดับเรขาคณิต (หาพจน์ที่ n)

คำถาม: ลำดับเรขาคณิตมีพจน์แรกคือ 3 และอัตราส่วนร่วมคือ 2 จงหาพจน์ที่ 5 ของลำดับนี้

แนวคิด

  • ระบุ $a_1$, $r$, และ $n$
  • ใช้สูตร $a_n = a_1 r^{n-1}$

เฉลย

จากโจทย์: $a_1 = 3$, $r = 2$, $n = 5$

ใช้สูตรพจน์ที่ $n$: $a_n = a_1 r^{n-1}$

$a_5 = 3 \cdot 2^{5-1}$

$a_5 = 3 \cdot 2^4$

$a_5 = 3 \cdot 16$

$a_5 = 48$

คำตอบ: 48

โจทย์ข้อที่ 4: ลำดับเรขาคณิต (หาผลบวก n พจน์แรก)

คำถาม: จงหาผลบวกของ 6 พจน์แรกของลำดับเรขาคณิต $1, 2, 4, 8, \ldots$

แนวคิด

  • ระบุ $a_1$, $r$, และ $n$
  • ใช้สูตร $S_n = \frac{a_1(r^n - 1)}{r - 1}$

เฉลย

จากลำดับ: $a_1 = 1$

หาอัตราส่วนร่วม $r = \frac{2}{1} = 2$

ต้องการหาผลบวก 6 พจน์แรก ดังนั้น $n = 6$

ใช้สูตร $S_n = \frac{a_1(r^n - 1)}{r - 1}$

$S_6 = \frac{1(2^6 - 1)}{2 - 1}$

$S_6 = \frac{1(64 - 1)}{1}$

$S_6 = 63$

คำตอบ: 63

โจทย์ข้อที่ 5: การประยุกต์ใช้ลำดับ (ระบุประเภท)

คำถาม: ปัจจุบันมีแบคทีเรีย 100 เซลล์ หากแบคทีเรียเพิ่มจำนวนเป็น 2 เท่าทุกๆ ชั่วโมง จงหาจำนวนแบคทีเรียเมื่อเวลาผ่านไป 5 ชั่วโมง (ไม่รวมจำนวนเริ่มต้น)

แนวคิด

  • ระบุว่าเป็นลำดับประเภทใด (เพิ่มเป็นเท่าตัวแสดงว่าเป็นเรขาคณิต)
  • ระบุ $a_1$ และ $r$ และ $n$ ที่ถูกต้อง
  • (เมื่อเวลาผ่านไป 1 ชั่วโมง คือพจน์ที่ 2)
  • (เมื่อเวลาผ่านไป 5 ชั่วโมง คือพจน์ที่ 6)

เฉลย

เป็นปัญหาลำดับเรขาคณิต

จำนวนแบคทีเรียเริ่มต้น (ที่เวลา 0 ชั่วโมง) คือ $a_1 = 100$ เซลล์

อัตราส่วนร่วม $r = 2$ (เพิ่มเป็น 2 เท่า)

ต้องการหาจำนวนแบคทีเรียเมื่อเวลาผ่านไป 5 ชั่วโมง นั่นคือพจน์ที่ $n = 5 + 1 = 6$

ใช้สูตร $a_n = a_1 r^{n-1}$

$a_6 = 100 \cdot 2^{6-1}$

$a_6 = 100 \cdot 2^5$

$a_6 = 100 \cdot 32$

$a_6 = 3,200$ เซลล์

คำตอบ: 3,200 เซลล์

8.5.4 ตัวอย่างข้อสอบฝึกทำ: ลำดับเลขคณิตและลำดับเรขาคณิต

เอกสารนี้รวบรวมตัวอย่างข้อสอบสำหรับนักเรียนได้ฝึกทำในหัวข้อ "ลำดับเลขคณิตและลำดับเรขาคณิต" เพื่อเสริมสร้างความเข้าใจและเตรียมความพร้อมสำหรับการสอบ สอวน. คอมพิวเตอร์ครับ

โจทย์ข้อที่ 1: ลำดับเลขคณิต (หาพจน์ที่ n)

คำถาม: ลำดับเลขคณิตมีพจน์แรกคือ 7 และผลต่างร่วมคือ 4 จงหาพจน์ที่ 12 ของลำดับนี้

แนวคิด

  • ระบุ $a_1$, $d$, และ $n$
  • ใช้สูตร $a_n = a_1 + (n-1)d$

เฉลย

จากโจทย์: $a_1 = 7$, $d = 4$, $n = 12$

ใช้สูตรพจน์ที่ $n$: $a_n = a_1 + (n-1)d$

$a_{12} = 7 + (12-1)4$

$a_{12} = 7 + (11)4$

$a_{12} = 7 + 44$

$a_{12} = 51$

คำตอบ: 51

โจทย์ข้อที่ 2: ลำดับเลขคณิต (หาผลบวก n พจน์แรก)

คำถาม: จงหาผลบวกของ 20 พจน์แรกของลำดับเลขคณิต $3, 7, 11, 15, \ldots$

แนวคิด

  • ระบุ $a_1$, $d$, และ $n$
  • ใช้สูตร $S_n = \frac{n}{2}(2a_1 + (n-1)d)$

เฉลย

จากลำดับ: $a_1 = 3$

หาผลต่างร่วม $d = 7 - 3 = 4$

ต้องการหาผลบวก 20 พจน์แรก ดังนั้น $n = 20$

ใช้สูตร $S_n = \frac{n}{2}(2a_1 + (n-1)d)$

$S_{20} = \frac{20}{2}(2(3) + (20-1)4)$

$S_{20} = 10(6 + (19)4)$

$S_{20} = 10(6 + 76)$

$S_{20} = 10(82)$

$S_{20} = 820$

คำตอบ: 820

โจทย์ข้อที่ 3: ลำดับเรขาคณิต (หาพจน์ที่ n)

คำถาม: ลำดับเรขาคณิตมีพจน์แรกคือ 5 และอัตราส่วนร่วมคือ 3 จงหาพจน์ที่ 4 ของลำดับนี้

แนวคิด

  • ระบุ $a_1$, $r$, และ $n$
  • ใช้สูตร $a_n = a_1 r^{n-1}$

เฉลย

จากโจทย์: $a_1 = 5$, $r = 3$, $n = 4$

ใช้สูตรพจน์ที่ $n$: $a_n = a_1 r^{n-1}$

$a_4 = 5 \cdot 3^{4-1}$

$a_4 = 5 \cdot 3^3$

$a_4 = 5 \cdot 27$

$a_4 = 135$

คำตอบ: 135

โจทย์ข้อที่ 4: ลำดับเรขาคณิต (หาผลบวก n พจน์แรก)

คำถาม: จงหาผลบวกของ 5 พจน์แรกของลำดับเรขาคณิต $2, 6, 18, 54, \ldots$

แนวคิด

  • ระบุ $a_1$, $r$, และ $n$
  • ใช้สูตร $S_n = \frac{a_1(r^n - 1)}{r - 1}$

เฉลย

จากลำดับ: $a_1 = 2$

หาอัตราส่วนร่วม $r = \frac{6}{2} = 3$

ต้องการหาผลบวก 5 พจน์แรก ดังนั้น $n = 5$

ใช้สูตร $S_n = \frac{a_1(r^n - 1)}{r - 1}$

$S_5 = \frac{2(3^5 - 1)}{3 - 1}$

$S_5 = \frac{2(243 - 1)}{2}$

$S_5 = 242$

คำตอบ: 242

โจทย์ข้อที่ 5: การประยุกต์ใช้ลำดับ (ระบุประเภทและหาพจน์)

คำถาม: ปัจจุบันมีจำนวนประชากร 1,000 คน หากจำนวนประชากรเพิ่มขึ้น 10% ทุกปี จงหาจำนวนประชากรเมื่อสิ้นสุดปีที่ 3 (ไม่รวมจำนวนเริ่มต้น)

แนวคิด

  • ระบุว่าเป็นลำดับประเภทใด (เพิ่มเป็นเปอร์เซ็นต์แสดงว่าเป็นเรขาคณิต)
  • ระบุ $a_1$ (จำนวนเริ่มต้น) และ $r$ (อัตราส่วนการเพิ่ม) และ $n$ (พจน์ที่ต้องการ)
  • $r = 1 + $ อัตราการเพิ่ม
  • สิ้นสุดปีที่ 3 หมายถึง พจน์ที่ 4

เฉลย

เป็นปัญหาลำดับเรขาคณิต

จำนวนประชากรเริ่มต้น (ที่เวลา 0 ปี) คือ $a_1 = 1,000$ คน

อัตราส่วนร่วม $r = 1 + 10\% = 1 + 0.10 = 1.10$

ต้องการหาจำนวนประชากรเมื่อสิ้นสุดปีที่ 3 นั่นคือพจน์ที่ $n = 3 + 1 = 4$

ใช้สูตร $a_n = a_1 r^{n-1}$

$a_4 = 1000 \cdot (1.10)^{4-1}$

$a_4 = 1000 \cdot (1.10)^3$

$a_4 = 1000 \cdot 1.331$

$a_4 = 1,331$ คน

คำตอบ: 1,331 คน

โจทย์ข้อที่ 6: ลำดับเลขคณิต (หา d และ $a_1$ จากพจน์)

คำถาม: ลำดับเลขคณิตมีพจน์ที่ 3 เท่ากับ 12 และพจน์ที่ 7 เท่ากับ 28 จงหาพจน์แรก ($a_1$) และผลต่างร่วม ($d$)

แนวคิด

  • ตั้งสมการจากสูตร $a_n = a_1 + (n-1)d$ สำหรับแต่ละพจน์ที่กำหนด
  • แก้ระบบสมการเชิงเส้นสองตัวแปร

เฉลย

1. พจน์ที่ 3 เท่ากับ 12: $a_3 = a_1 + (3-1)d \Rightarrow a_1 + 2d = 12$ ...(1)

2. พจน์ที่ 7 เท่ากับ 28: $a_7 = a_1 + (7-1)d \Rightarrow a_1 + 6d = 28$ ...(2)

นำสมการ (2) ลบด้วยสมการ (1):

$(a_1 + 6d) - (a_1 + 2d) = 28 - 12$

$4d = 16$

$d = 4$

แทนค่า $d = 4$ ลงในสมการ (1):

$a_1 + 2(4) = 12$

$a_1 + 8 = 12$

$a_1 = 4$

คำตอบ: $a_1 = 4$, $d = 4$

โจทย์ข้อที่ 7: ลำดับเรขาคณิต (หา r และ $a_1$ จากพจน์)

คำถาม: ลำดับเรขาคณิตมีพจน์ที่ 2 เท่ากับ 10 และพจน์ที่ 4 เท่ากับ 250 จงหาอัตราส่วนร่วม ($r$) และพจน์แรก ($a_1$)

แนวคิด

  • ตั้งสมการจากสูตร $a_n = a_1 r^{n-1}$ สำหรับแต่ละพจน์ที่กำหนด
  • แก้ระบบสมการ (โดยการหารสมการ)

เฉลย

1. พจน์ที่ 2 เท่ากับ 10: $a_2 = a_1 r^{2-1} \Rightarrow a_1 r = 10$ ...(1)

2. พจน์ที่ 4 เท่ากับ 250: $a_4 = a_1 r^{4-1} \Rightarrow a_1 r^3 = 250$ ...(2)

นำสมการ (2) หารด้วยสมการ (1):

$\frac{a_1 r^3}{a_1 r} = \frac{250}{10}$

$r^2 = 25$

$r = \pm 5$

กรณี $r = 5$: แทนใน (1) $a_1(5) = 10 \Rightarrow a_1 = 2$

กรณี $r = -5$: แทนใน (1) $a_1(-5) = 10 \Rightarrow a_1 = -2$

คำตอบ: $r = 5, a_1 = 2$ หรือ $r = -5, a_1 = -2$

โจทย์ข้อที่ 8: การหาสมาชิกของลำดับ (เมื่อกำหนดผลรวม)

คำถาม: ลำดับเลขคณิตมีผลบวก 4 พจน์แรกเท่ากับ 28 และผลต่างร่วมคือ 3 จงหาพจน์แรกของลำดับนี้

แนวคิด

  • ใช้สูตรผลบวก $S_n = \frac{n}{2}(2a_1 + (n-1)d)$ และแก้สมการหา $a_1$

เฉลย

จากโจทย์: $S_4 = 28$, $n = 4$, $d = 3$

ใช้สูตร $S_n = \frac{n}{2}(2a_1 + (n-1)d)$

$28 = \frac{4}{2}(2a_1 + (4-1)3)$

$28 = 2(2a_1 + 3 \cdot 3)$

$28 = 2(2a_1 + 9)$

หารด้วย 2 ทั้งสองข้าง:

$14 = 2a_1 + 9$

$2a_1 = 14 - 9$

$2a_1 = 5$

$a_1 = \frac{5}{2}$

คำตอบ: $a_1 = \frac{5}{2}$

โจทย์ข้อที่ 9: การหาจำนวนพจน์ (เมื่อกำหนดผลรวม)

คำถาม: ลำดับเรขาคณิตมีพจน์แรกเป็น 1 และอัตราส่วนร่วมเป็น 2 ถ้าผลบวกของลำดับนี้คือ 127 จงหาว่าลำดับนี้มีกี่พจน์

แนวคิด

  • ใช้สูตรผลบวก $S_n = \frac{a_1(r^n - 1)}{r - 1}$ และแก้สมการหา $n$

เฉลย

จากโจทย์: $a_1 = 1$, $r = 2$, $S_n = 127$

ใช้สูตร $S_n = \frac{a_1(r^n - 1)}{r - 1}$

$127 = \frac{1(2^n - 1)}{2 - 1}$

$127 = \frac{2^n - 1}{1}$

$127 = 2^n - 1$

$128 = 2^n$

$2^7 = 2^n$

ดังนั้น $n = 7$

คำตอบ: 7 พจน์

โจทย์ข้อที่ 10: โจทย์ปัญหาประยุกต์ผสม

คำถาม: นาย A เริ่มฝากเงิน 100 บาทในเดือนแรก และจะฝากเพิ่มขึ้นเดือนละ 20 บาทในเดือนถัดๆ ไป (เช่น เดือนที่สองฝาก 120 บาท) จงหาว่าเมื่อสิ้นสุดเดือนที่ 12 นาย A จะมีเงินฝากรวมทั้งหมดเท่าไหร่?

แนวคิด

  • ปัญหานี้เป็นลำดับเลขคณิต (เพิ่มขึ้นคงที่)
  • ระบุ $a_1$, $d$, $n$
  • ใช้สูตรผลบวก $S_n = \frac{n}{2}(2a_1 + (n-1)d)$

เฉลย

เป็นปัญหาลำดับเลขคณิต

พจน์แรก ($a_1$): เงินฝากเดือนแรก = 100 บาท

ผลต่างร่วม ($d$): เงินฝากเพิ่มขึ้นเดือนละ 20 บาท = 20 บาท

จำนวนพจน์ ($n$): สิ้นสุดเดือนที่ 12 = 12 เดือน

ใช้สูตร $S_n = \frac{n}{2}(2a_1 + (n-1)d)$

$S_{12} = \frac{12}{2}(2(100) + (12-1)20)$

$S_{12} = 6(200 + (11)20)$

$S_{12} = 6(200 + 220)$

$S_{12} = 6(420)$

$S_{12} = 2,520$

คำตอบ: 2,520 บาท
