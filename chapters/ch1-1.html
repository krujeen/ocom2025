1.1.1 ระบบจำนวนจริง (Real Number System)

นิยาม

จำนวนจริง ($\mathbb{R}$) คือ จำนวนทุกชนิดที่สามารถแทนได้ด้วยจุดบนเส้นจำนวน (Number Line)

ประเภทของจำนวนจริง

1. จำนวนตรรกยะ (Rational Numbers: $\mathbb{Q}$)

จำนวนที่สามารถเขียนในรูปเศษส่วน $\frac{a}{b}$ ได้ โดยที่ $a$ และ $b$ เป็นจำนวนเต็ม และ $b \neq 0$

จำนวนเต็ม (Integers: $\mathbb{Z}$)

$\ldots, -3, -2, -1, 0, 1, 2, 3, \ldots$

  • จำนวนเต็มลบ: $-1, -2, -3, \ldots$
  • ศูนย์: $0$
  • จำนวนเต็มบวก (หรือ จำนวนนับ: $\mathbb{N}$): $1, 2, 3, \ldots$
เศษส่วนและทศนิยมซ้ำ

ตัวอย่าง: $\frac{1}{2}$, $0.5$, $-3\frac{1}{4}$, $0.333\ldots$

2. จำนวนอตรรกยะ (Irrational Numbers: $\mathbb{Q}'$)

จำนวนที่ไม่สามารถเขียนในรูปเศษส่วน $\frac{a}{b}$ ได้ และมีค่าเป็นทศนิยมไม่ซ้ำไม่รู้จบ

ตัวอย่าง: $\sqrt{2}$, $\pi$ (พาย), $e$ (ประมาณ 2.718), รากที่ไม่ลงตัว (เช่น $\sqrt{3}$, $\sqrt[3]{5}$)

แผนภาพแสดงระบบจำนวนจริง

1.1.2 สมบัติของจำนวนจริง

ก. สมบัติการเท่ากันของจำนวนจริง

สมบัติการสะท้อน (Reflexive Property)

$a = a$ (เช่น $5 = 5$)

สมบัติการสมมาตร (Symmetric Property)

ถ้า $a = b$ แล้ว $b = a$ (เช่น ถ้า $x = 3$ แล้ว $3 = x$)

สมบัติการถ่ายทอด (Transitive Property)

ถ้า $a = b$ และ $b = c$ แล้ว $a = c$ (เช่น ถ้า $x = y$ และ $y = 7$ แล้ว $x = 7$)

สมบัติการบวกด้วยจำนวนที่เท่ากัน (Additive Property)

ถ้า $a = b$ แล้ว $a + c = b + c$

สมบัติการคูณด้วยจำนวนที่เท่ากัน (Multiplicative Property)

ถ้า $a = b$ แล้ว $ac = bc$

สมบัติการแจกแจง (Distributive Property)

$a(b + c) = ab + ac$ และ $(a + b)c = ac + bc$

ข. สมบัติเกี่ยวกับการบวกและการคูณของจำนวนจริง

ให้ $a, b, c$ เป็นจำนวนจริง

สมบัติการปิด (Closure Property)

  • การบวก: $a + b$ เป็นจำนวนจริง
  • การคูณ: $ab$ เป็นจำนวนจริง

สมบัติการสลับที่ (Commutative Property)

  • การบวก: $a + b = b + a$
  • การคูณ: $ab = ba$

สมบัติการเปลี่ยนกลุ่ม (Associative Property)

  • การบวก: $(a + b) + c = a + (b + c)$
  • การคูณ: $(ab)c = a(bc)$

การมีเอกลักษณ์ (Identity Element)

  • เอกลักษณ์การบวก: $0$ เพราะ $a + 0 = a = 0 + a$
  • เอกลักษณ์การคูณ: $1$ เพราะ $a \cdot 1 = a = 1 \cdot a$

การมีอินเวอร์ส (Inverse Element)

  • อินเวอร์สการบวก: $-a$ เพราะ $a + (-a) = 0 = (-a) + a$
  • อินเวอร์สการคูณ: $a^{-1}$ หรือ $\frac{1}{a}$ เพราะ $a \cdot a^{-1} = 1 = a^{-1} \cdot a$ (โดยที่ $a \neq 0$)

1.1.3 ค่าสัมบูรณ์ (Absolute Value)

ความหมาย

ค่าสัมบูรณ์ของจำนวนจริง $a$ เขียนแทนด้วย $|a|$ คือระยะทางจาก $0$ ถึง $a$ บนเส้นจำนวนเสมอ ซึ่งค่าที่ได้จะเป็นบวกหรือศูนย์เสมอ

นิยาม

$$|a| = \begin{cases} a & \text{เมื่อ } a \geq 0 \\ -a & \text{เมื่อ } a < 0 \end{cases}$$

ตัวอย่าง

  • $|5| = 5$
  • $|-5| = 5$
  • $|0| = 0$

1.1.4 จำนวนจริงในรูปกรณฑ์ (Radicals) และเลขยกกำลัง (Exponents)

กรณฑ์ที่ n (n-th Root)

$\sqrt[n]{a}$ หมายถึง รากที่ $n$ ของ $a$

กฎสำคัญ

  • ถ้า $n$ เป็นจำนวนคู่และ $a \geq 0$, $\sqrt[n]{a}$ จะมีค่าเดียวคือค่าที่ไม่เป็นลบ
  • ถ้า $n$ เป็นจำนวนคี่, $\sqrt[n]{a}$ จะมีค่าเดียว

คุณสมบัติสำคัญ

  • $\sqrt[n]{a^n} = |a|$ เมื่อ $n$ เป็นจำนวนคู่
  • $\sqrt[n]{a^n} = a$ เมื่อ $n$ เป็นจำนวนคี่

เลขยกกำลัง (Exponents): $a^n$

สมบัติสำคัญ

  • $a^m \cdot a^n = a^{m+n}$
  • $\frac{a^m}{a^n} = a^{m-n}$
  • $(a^m)^n = a^{mn}$
  • $(ab)^n = a^n b^n$
  • $\left(\frac{a}{b}\right)^n = \frac{a^n}{b^n}$
  • $a^0 = 1$ ($a \neq 0$)
  • $a^{-n} = \frac{1}{a^n}$ ($a \neq 0$)
  • $a^{\frac{m}{n}} = \sqrt[n]{a^m}$

1.1.5 ตัวอย่างโจทย์และแนวทางการแก้ปัญหา

การทำความเข้าใจเรื่องจำนวนจริงและสมบัติจะช่วยให้แก้ปัญหาคณิตศาสตร์พื้นฐานได้อย่างแม่นยำ รวมถึงเป็นพื้นฐานสำคัญสำหรับหัวข้อที่ซับซ้อนขึ้นในวิทยาการคำนวณ เช่น การวิเคราะห์โดเมน/เรนจ์ของฟังก์ชัน, การจัดการข้อมูลเชิงตัวเลข, และการวิเคราะห์เงื่อนไขต่างๆ ในโปรแกรม

ตัวอย่างที่ 1: การจำแนกประเภทจำนวน

โจทย์

จงจำแนกจำนวนต่อไปนี้ว่าเป็นจำนวนชนิดใด: $\sqrt{9}$, $-3.14$, $\frac{22}{7}$, $\sqrt{5}$, $0$, $0.666\ldots$, $\frac{\pi}{2}$, $100$

แนวคิด

พิจารณาจากนิยามของจำนวนแต่ละประเภท

  • $\sqrt{9} = 3$
  • $0.666\ldots = \frac{2}{3}$

เฉลย

  • จำนวนนับ ($\mathbb{N}$): $100$
  • จำนวนเต็ม ($\mathbb{Z}$): $\sqrt{9}$ ($=3$), $0$, $100$
  • จำนวนตรรกยะ ($\mathbb{Q}$): $\sqrt{9}$ ($=3$), $-3.14$, $\frac{22}{7}$, $0$, $0.666\ldots$, $100$
  • จำนวนอตรรกยะ ($\mathbb{Q}'$): $\sqrt{5}$, $\frac{\pi}{2}$
  • จำนวนจริง ($\mathbb{R}$): ทุกจำนวนที่กำหนดให้

ตัวอย่างที่ 2: การใช้สมบัติค่าสัมบูรณ์

โจทย์

จงหาค่าของ $|2 - \sqrt{7}| + |1 - \sqrt{7}|$

แนวคิด

  1. ประเมินค่าของ $\sqrt{7}$ (ประมาณ $2.64$)
  2. พิจารณาเครื่องหมายของนิพจน์ภายในค่าสัมบูรณ์แต่ละตัว
  3. $2 - \sqrt{7}: 2 - 2.64 = -0.64$ (ติดลบ)
  4. $1 - \sqrt{7}: 1 - 2.64 = -1.64$ (ติดลบ)
  5. ใช้คุณสมบัติของค่าสัมบูรณ์: $|a| = -a$ ถ้า $a < 0$

เฉลย

$|2 - \sqrt{7}| = -(2 - \sqrt{7}) = -2 + \sqrt{7}$ (เนื่องจาก $2 < \sqrt{7}$)

$|1 - \sqrt{7}| = -(1 - \sqrt{7}) = -1 + \sqrt{7}$ (เนื่องจาก $1 < \sqrt{7}$)

ดังนั้น:

$$|2 - \sqrt{7}| + |1 - \sqrt{7}| = (-2 + \sqrt{7}) + (-1 + \sqrt{7})$$

$$= -2 + \sqrt{7} - 1 + \sqrt{7} = -3 + 2\sqrt{7}$$

ตัวอย่างที่ 3: การใช้สมบัติเลขยกกำลังและกรณฑ์

โจทย์

จงหาค่าของ $\frac{(2^3 \cdot 3^2)^2}{6^3} \cdot \sqrt[3]{-125}$

แนวคิด

  1. ใช้สมบัติเลขยกกำลัง: $(ab)^n = a^n b^n$ และ $(a^m)^n = a^{mn}$
  2. แปลงฐานให้เป็นเลขเดียวกัน (ถ้าเป็นไปได้): $6^3 = (2 \cdot 3)^3 = 2^3 \cdot 3^3$
  3. หาค่าของกรณฑ์ที่สาม: $\sqrt[3]{-125}$

เฉลย

$$\frac{(2^3 \cdot 3^2)^2}{6^3} \cdot \sqrt[3]{-125}$$

$$= \frac{2^{3 \cdot 2} \cdot 3^{2 \cdot 2}}{(2 \cdot 3)^3} \cdot (-5)$$

$$= \frac{2^6 \cdot 3^4}{2^3 \cdot 3^3} \cdot (-5)$$

$$= (2^{6-3} \cdot 3^{4-3}) \cdot (-5)$$

$$= (2^3 \cdot 3^1) \cdot (-5)$$

$$= (8 \cdot 3) \cdot (-5) = 24 \cdot (-5) = -120$$

ตัวอย่างที่ 4: การแก้ปัญหาผสมผสาน

โจทย์

ถ้า $x$ เป็นจำนวนเต็มบวก และ $\sqrt{256^x} = 2^{x+6}$ จงหาค่าของ $x$

แนวคิด

  1. แปลงฐานของเลขยกกำลังให้เป็นฐานเดียวกัน (ในที่นี้คือฐาน $2$)
  2. $256 = 2^8$
  3. ใช้สมบัติเลขยกกำลังและกรณฑ์: $\sqrt{a^m} = a^{\frac{m}{2}}$
  4. แก้สมการเลขยกกำลัง

เฉลย

$$\sqrt{256^x} = 2^{x+6}$$

$$\sqrt{(2^8)^x} = 2^{x+6}$$

$$\sqrt{2^{8x}} = 2^{x+6}$$

$$2^{\frac{8x}{2}} = 2^{x+6}$$

$$2^{4x} = 2^{x+6}$$

เนื่องจากฐานเท่ากัน เลขชี้กำลังจึงต้องเท่ากัน:

$$4x = x + 6$$

$$4x - x = 6$$

$$3x = 6$$

$$x = 2$$

1.1.6 ตัวอย่างข้อสอบฝึกทำ

เอกสารนี้รวบรวมตัวอย่างข้อสอบสำหรับนักเรียนได้ฝึกทำในหัวข้อ "จำนวนจริงและสมบัติ" เพื่อเสริมสร้างความเข้าใจและเตรียมความพร้อมสำหรับการสอบ สอวน. คอมพิวเตอร์

โจทย์ข้อที่ 1: การจำแนกประเภทจำนวน

คำถาม

จงพิจารณาว่าข้อความใดต่อไปนี้ เป็นจริง

  • ก. จำนวนเต็มที่มากที่สุดที่น้อยกว่า $\sqrt{10}$ คือ 3
  • ข. $0.121221222\ldots$ เป็นจำนวนตรรกยะ
  • ค. ผลบวกของจำนวนอตรรกยะสองจำนวนต้องเป็นจำนวนอตรรกยะเสมอ
  • ง. จำนวนจริงที่ไม่ใช่จำนวนตรรกยะคือจำนวนอตรรกยะ

แนวคิด

  • ก. หาค่าประมาณของ $\sqrt{10}$ แล้วพิจารณาจำนวนเต็มที่อยู่ใกล้เคียง
  • ข. พิจารณารูปแบบของทศนิยมว่าซ้ำหรือไม่
  • ค. ทดสอบด้วยตัวอย่าง เช่น $\sqrt{2} + (-\sqrt{2})$
  • ง. พิจารณาแผนภาพระบบจำนวนจริง

เฉลย

ก. $\sqrt{10} \approx 3.16$ จำนวนเต็มที่มากที่สุดที่น้อยกว่า $3.16$ คือ $3$ ดังนั้นข้อนี้เป็นจริง

ข. $0.121221222\ldots$ เป็นทศนิยมไม่ซ้ำไม่รู้จบ (มีรูปแบบที่เพิ่มจำนวนเลข 2) ดังนั้นเป็นจำนวนอตรรกยะ ข้อนี้จึงเป็นเท็จ

ค. เช่น $\sqrt{2}$ เป็นจำนวนอตรรกยะ และ $2-\sqrt{2}$ เป็นจำนวนอตรรกยะ แต่ $\sqrt{2} + (2-\sqrt{2}) = 2$ ซึ่งเป็นจำนวนตรรกยะ ดังนั้นข้อนี้เป็นเท็จ

ง. ตามนิยามและแผนภาพของระบบจำนวนจริง ข้อนี้เป็นจริง

คำตอบ: ก. และ ง.

โจทย์ข้อที่ 2: การใช้สมบัติค่าสัมบูรณ์และกรณฑ์

คำถาม

กำหนดให้ $x = \sqrt{17}$ จงหาค่าของ $|4-x| - |x-5|$

แนวคิด

  • ประเมินค่าของ $\sqrt{17}$ (ประมาณ 4.12)
  • พิจารณาเครื่องหมายของนิพจน์ภายในค่าสัมบูรณ์แต่ละตัวเทียบกับ $x$
  • ใช้คุณสมบัติของค่าสัมบูรณ์: $|a| = a$ ถ้า $a \geq 0$ และ $|a| = -a$ ถ้า $a < 0$

เฉลย

ให้ $x = \sqrt{17}$ ซึ่ง $\sqrt{16} < \sqrt{17} < \sqrt{25}$ ดังนั้น $4 < \sqrt{17} < 5$

สำหรับ $|4-x|$: เนื่องจาก $4 < \sqrt{17}$ แสดงว่า $4-x$ มีค่าเป็นลบ
ดังนั้น $|4-x| = -(4-x) = -4+x$

สำหรับ $|x-5|$: เนื่องจาก $\sqrt{17} < 5$ แสดงว่า $x-5$ มีค่าเป็นลบ
ดังนั้น $|x-5| = -(x-5) = -x+5$

นำค่าที่ได้มาแทนในนิพจน์:

$$|4-x| - |x-5| = (-4+x) - (-x+5) = -4+x+x-5 = 2x-9$$

แทนค่า $x = \sqrt{17}$ กลับเข้าไป: $= 2\sqrt{17} - 9$

คำตอบ: $2\sqrt{17} - 9$

โจทย์ข้อที่ 3: สมบัติเลขยกกำลัง

คำถาม

จงหาค่าของ $\frac{(27^n)^2}{3^{2n} \cdot (9^n)^2}$

แนวคิด

  • แปลงฐานของเลขยกกำลังทั้งหมดให้เป็นฐานเดียวกัน (ในที่นี้คือฐาน 3)
  • $9 = 3^2$
  • $27 = 3^3$
  • ใช้สมบัติเลขยกกำลัง $(a^m)^n = a^{mn}$ และ $\frac{a^n}{a^m} = a^{m-n}$

เฉลย

$$\frac{(27^n)^2}{3^{2n} \cdot (9^n)^2} = \frac{((3^3)^n)^2}{3^{2n} \cdot ((3^2)^n)^2}$$

$$= \frac{(3^{3n})^2}{3^{2n} \cdot (3^{2n})^2} = \frac{3^{6n}}{3^{2n} \cdot 3^{4n}}$$

$$= \frac{3^{6n}}{3^{2n+4n}} = \frac{3^{6n}}{3^{6n}} = 3^{6n-6n} = 3^0 = 1$$

คำตอบ: 1

โจทย์ข้อที่ 4: การแก้ปัญหาสมบัติจำนวนจริง

คำถาม

ถ้า $a, b$ เป็นจำนวนจริงที่ไม่เท่ากับศูนย์ และ $a^2 = b^2$ ข้อใดต่อไปนี้เป็นจริงเสมอ?

  • ก. $a = b$
  • ข. $a = -b$
  • ค. $a = b$ หรือ $a = -b$
  • ง. $a^3 = b^3$

แนวคิด

  • พิจารณาสมการ $a^2 = b^2$ แล้วแยกตัวประกอบ
  • พิจารณาความเป็นไปได้ของ $a$ และ $b$
  • ตรวจสอบแต่ละตัวเลือก

เฉลย

จาก $a^2 = b^2$

ย้ายข้าง: $a^2 - b^2 = 0$

แยกตัวประกอบ: $(a-b)(a+b) = 0$

ดังนั้น $a-b = 0$ หรือ $a+b = 0$

ซึ่งหมายความว่า $a = b$ หรือ $a = -b$

ก. $a = b$: ไม่จริงเสมอไป เช่น ถ้า $a = 2, b = -2$ จะได้ $2^2 = (-2)^2$ แต่ $2 \neq -2$

ข. $a = -b$: ไม่จริงเสมอไป เช่น ถ้า $a = 2, b = 2$ จะได้ $2^2 = 2^2$ แต่ $2 \neq -2$

ค. $a = b$ หรือ $a = -b$: เป็นจริงเสมอ ตามการวิเคราะห์ข้างต้น

ง. $a^3 = b^3$: ไม่จริงเสมอไป เช่น ถ้า $a = 2, b = -2$ จะได้ $2^3 = 8$ และ $(-2)^3 = -8$ ซึ่ง $8 \neq -8$

คำตอบ: ค.

โจทย์ข้อที่ 5: การประยุกต์ใช้สมบัติค่าสัมบูรณ์ในอสมการ

คำถาม

เซตคำตอบของอสมการ $|x-3| < 5$ คือข้อใด?

  • ก. $(-\infty, 8)$
  • ข. $(-2, 8)$
  • ค. $(-2, -\infty)$
  • ง. $(-\infty, -2) \cup (8, \infty)$

แนวคิด

  • ใช้นิยามของค่าสัมบูรณ์ในอสมการ: ถ้า $|u| < c$ แล้ว $-c < u < c$
  • แก้อสมการเพื่อหาช่วงของ $x$

เฉลย

จากอสมการ $|x-3| < 5$

ตามนิยามของค่าสัมบูรณ์:

$$-5 < x-3 < 5$$

บวก 3 ตลอดอสมการ:

$$-5+3 < x-3+3 < 5+3$$

$$-2 < x < 8$$

ดังนั้น เซตคำตอบคือช่วง $(-2, 8)$

คำตอบ: ข.

โจทย์ข้อที่ 6: การประยุกต์ใช้สมบัติเลขยกกำลังและกรณฑ์

คำถาม

ถ้า $2^{x+1} + 2^{x+2} + 2^{x+3} = 112$ จงหาค่าของ $x$

แนวคิด

  • ดึงตัวร่วมของเลขยกกำลังที่มีฐานเดียวกัน
  • แก้อสมการเพื่อหาค่า $x$

เฉลย

$$2^{x+1} + 2^{x+2} + 2^{x+3} = 112$$

$$2^{x+1}(1 + 2^1 + 2^2) = 112$$

$$2^{x+1}(1 + 2 + 4) = 112$$

$$2^{x+1}(7) = 112$$

$$2^{x+1} = \frac{112}{7} = 16$$

$$2^{x+1} = 2^4$$

เนื่องจากฐานเท่ากัน เลขชี้กำลังจึงต้องเท่ากัน:

$$x+1 = 4$$

$$x = 3$$

คำตอบ: $x = 3$

โจทย์ข้อที่ 7: การใช้สมบัติการสลับที่และเปลี่ยนกลุ่ม

คำถาม

จงหาค่าของ $(123 \cdot 456) + (789 \cdot 123) - (123 \cdot 245)$ โดยใช้สมบัติการแจกแจง

แนวคิด

  • สังเกตว่ามีตัวร่วมคือ 123
  • ใช้สมบัติการแจกแจง $a(b+c) = ab + ac$ หรือ $ab + ac - ad = a(b + c - d)$

เฉลย

$$(123 \cdot 456) + (789 \cdot 123) - (123 \cdot 245)$$

$$= 123 \cdot (456 + 789 - 245)$$

$$= 123 \cdot (1245 - 245)$$

$$= 123 \cdot (1000)$$

$$= 123000$$

คำตอบ: 123000

โจทย์ข้อที่ 8: การหาค่าสัมบูรณ์ของนิพจน์ที่ซับซ้อนขึ้น

คำถาม

ถ้า $a = \frac{1}{\sqrt{3} - \sqrt{2}}$ และ $b = \frac{1}{\sqrt{3} + \sqrt{2}}$ จงหาค่าของ $|a - b|$

แนวคิด

  • ทำส่วนให้เป็นจำนวนเต็มโดยการคูณด้วยคอนจูเกต
  • แทนค่า $a$ และ $b$ ลงในนิพจน์ $|a - b|$
  • พิจารณาค่าภายในค่าสัมบูรณ์

เฉลย

หาค่า $a$:

$$a = \frac{1}{\sqrt{3} - \sqrt{2}} \cdot \frac{\sqrt{3} + \sqrt{2}}{\sqrt{3} + \sqrt{2}} = \frac{\sqrt{3} + \sqrt{2}}{(\sqrt{3})^2 - (\sqrt{2})^2} = \frac{\sqrt{3} + \sqrt{2}}{3 - 2} = \sqrt{3} + \sqrt{2}$$

หาค่า $b$:

$$b = \frac{1}{\sqrt{3} + \sqrt{2}} \cdot \frac{\sqrt{3} - \sqrt{2}}{\sqrt{3} - \sqrt{2}} = \frac{\sqrt{3} - \sqrt{2}}{(\sqrt{3})^2 - (\sqrt{2})^2} = \frac{\sqrt{3} - \sqrt{2}}{3 - 2} = \sqrt{3} - \sqrt{2}$$

หาค่า $|a - b|$:

$$|a - b| = |(\sqrt{3} + \sqrt{2}) - (\sqrt{3} - \sqrt{2})| = |\sqrt{3} + \sqrt{2} - \sqrt{3} + \sqrt{2}| = |2\sqrt{2}|$$

เนื่องจาก $2\sqrt{2}$ เป็นจำนวนบวก: $= 2\sqrt{2}$

คำตอบ: $2\sqrt{2}$

โจทย์ข้อที่ 9: การจำแนกประเภทจำนวนที่ซับซ้อน

คำถาม

ข้อใดต่อไปนี้เป็นจำนวนอตรรกยะ?

  • ก. $\sqrt[3]{-8}$
  • ข. $\sqrt{2.25}$
  • ค. $\frac{2\pi}{3\pi}$
  • ง. $\sqrt{12}$

แนวคิด

  • พิจารณาแต่ละตัวเลือกว่าสามารถเขียนในรูปเศษส่วนของจำนวนเต็มได้หรือไม่ หรือเป็นทศนิยมซ้ำหรือไม่
  • รากที่สองของจำนวนที่ไม่ใช่กำลังสองสมบูรณ์มักจะเป็นจำนวนอตรรกยะ

เฉลย

ก. $\sqrt[3]{-8} = -2$ ซึ่งเป็นจำนวนเต็ม ดังนั้นเป็นจำนวนตรรกยะ

ข. $\sqrt{2.25} = \sqrt{\frac{225}{100}} = \frac{15}{10} = \frac{3}{2}$ ซึ่งเป็นเศษส่วน ดังนั้นเป็นจำนวนตรรกยะ

ค. $\frac{2\pi}{3\pi} = \frac{2}{3}$ (เมื่อ $\pi \neq 0$) ซึ่งเป็นเศษส่วน ดังนั้นเป็นจำนวนตรรกยะ

ง. $\sqrt{12} = \sqrt{4 \cdot 3} = 2\sqrt{3}$ ซึ่งไม่สามารถเขียนในรูปเศษส่วนของจำนวนเต็มได้ (เนื่องจาก $\sqrt{3}$ เป็นจำนวนอตรรกยะ) ดังนั้นเป็นจำนวนอตรรกยะ

คำตอบ: ง.

โจทย์ข้อที่ 10: การประยุกต์ใช้สมบัติเลขยกกำลัง

คำถาม

ถ้า $2^{2x} = 5$ จงหาค่าของ $4^{x+1}$

แนวคิด

  • แปลงฐานของ $4^{x+1}$ ให้เป็นฐาน 2
  • ใช้สมบัติเลขยกกำลัง $(a^m)^n = a^{mn}$ และ $a^{m+n} = a^m \cdot a^n$
  • แทนค่า $2^{2x}$

เฉลย

จาก $2^{2x} = 5$

พิจารณา $4^{x+1}$:

$$4^{x+1} = (2^2)^{x+1} = 2^{2(x+1)} = 2^{2x+2} = 2^{2x} \cdot 2^2$$

เรารู้ว่า $2^{2x} = 5$ และ $2^2 = 4$

$$= 5 \cdot 4 = 20$$

คำตอบ: 20
