1.1.1 ระบบจำนวนจริง (Real Number System)

นิยาม

จำนวนจริง ($\mathbb{R}$) คือ จำนวนทุกชนิดที่สามารถแทนได้ด้วยจุดบนเส้นจำนวน (Number Line)

ประเภทของจำนวนจริง

1. จำนวนตรรกยะ (Rational Numbers: $\mathbb{Q}$)

จำนวนที่สามารถเขียนในรูปเศษส่วน $\frac{a}{b}$ ได้ โดยที่ $a$ และ $b$ เป็นจำนวนเต็ม และ $b \neq 0$

จำนวนเต็ม (Integers: $\mathbb{Z}$)

$\ldots, -3, -2, -1, 0, 1, 2, 3, \ldots$

  • จำนวนเต็มลบ: $-1, -2, -3, \ldots$
  • ศูนย์: $0$
  • จำนวนเต็มบวก (หรือ จำนวนนับ: $\mathbb{N}$): $1, 2, 3, \ldots$
เศษส่วนและทศนิยมซ้ำ

ตัวอย่าง: $\frac{1}{2}$, $0.5$, $-3\frac{1}{4}$, $0.333\ldots$

2. จำนวนอตรรกยะ (Irrational Numbers: $\mathbb{Q}'$)

จำนวนที่ไม่สามารถเขียนในรูปเศษส่วน $\frac{a}{b}$ ได้ และมีค่าเป็นทศนิยมไม่ซ้ำไม่รู้จบ

ตัวอย่าง: $\sqrt{2}$, $\pi$ (พาย), $e$ (ประมาณ 2.718), รากที่ไม่ลงตัว (เช่น $\sqrt{3}$, $\sqrt[3]{5}$)

แผนภาพแสดงระบบจำนวนจริง

1.1.2 สมบัติของจำนวนจริง

ก. สมบัติการเท่ากันของจำนวนจริง

สมบัติการสะท้อน (Reflexive Property)

$a = a$ (เช่น $5 = 5$)

สมบัติการสมมาตร (Symmetric Property)

ถ้า $a = b$ แล้ว $b = a$ (เช่น ถ้า $x = 3$ แล้ว $3 = x$)

สมบัติการถ่ายทอด (Transitive Property)

ถ้า $a = b$ และ $b = c$ แล้ว $a = c$ (เช่น ถ้า $x = y$ และ $y = 7$ แล้ว $x = 7$)

สมบัติการบวกด้วยจำนวนที่เท่ากัน (Additive Property)

ถ้า $a = b$ แล้ว $a + c = b + c$

สมบัติการคูณด้วยจำนวนที่เท่ากัน (Multiplicative Property)

ถ้า $a = b$ แล้ว $ac = bc$

สมบัติการแจกแจง (Distributive Property)

$a(b + c) = ab + ac$ และ $(a + b)c = ac + bc$

ข. สมบัติเกี่ยวกับการบวกและการคูณของจำนวนจริง

ให้ $a, b, c$ เป็นจำนวนจริง

สมบัติการปิด (Closure Property)

  • การบวก: $a + b$ เป็นจำนวนจริง
  • การคูณ: $ab$ เป็นจำนวนจริง

สมบัติการสลับที่ (Commutative Property)

  • การบวก: $a + b = b + a$
  • การคูณ: $ab = ba$

สมบัติการเปลี่ยนกลุ่ม (Associative Property)

  • การบวก: $(a + b) + c = a + (b + c)$
  • การคูณ: $(ab)c = a(bc)$

การมีเอกลักษณ์ (Identity Element)

  • เอกลักษณ์การบวก: $0$ เพราะ $a + 0 = a = 0 + a$
  • เอกลักษณ์การคูณ: $1$ เพราะ $a \cdot 1 = a = 1 \cdot a$

การมีอินเวอร์ส (Inverse Element)

  • อินเวอร์สการบวก: $-a$ เพราะ $a + (-a) = 0 = (-a) + a$
  • อินเวอร์สการคูณ: $a^{-1}$ หรือ $\frac{1}{a}$ เพราะ $a \cdot a^{-1} = 1 = a^{-1} \cdot a$ (โดยที่ $a \neq 0$)

1.1.3 ค่าสัมบูรณ์ (Absolute Value)

ความหมาย

ค่าสัมบูรณ์ของจำนวนจริง $a$ เขียนแทนด้วย $|a|$ คือระยะทางจาก $0$ ถึง $a$ บนเส้นจำนวนเสมอ ซึ่งค่าที่ได้จะเป็นบวกหรือศูนย์เสมอ

นิยาม

$$|a| = \begin{cases} a & \text{เมื่อ } a \geq 0 \\ -a & \text{เมื่อ } a < 0 \end{cases}$$

ตัวอย่าง

  • $|5| = 5$
  • $|-5| = 5$
  • $|0| = 0$

1.1.4 จำนวนจริงในรูปกรณฑ์ (Radicals) และเลขยกกำลัง (Exponents)

กรณฑ์ที่ n (n-th Root)

$\sqrt[n]{a}$ หมายถึง รากที่ $n$ ของ $a$

กฎสำคัญ

  • ถ้า $n$ เป็นจำนวนคู่และ $a \geq 0$, $\sqrt[n]{a}$ จะมีค่าเดียวคือค่าที่ไม่เป็นลบ
  • ถ้า $n$ เป็นจำนวนคี่, $\sqrt[n]{a}$ จะมีค่าเดียว

คุณสมบัติสำคัญ

  • $\sqrt[n]{a^n} = |a|$ เมื่อ $n$ เป็นจำนวนคู่
  • $\sqrt[n]{a^n} = a$ เมื่อ $n$ เป็นจำนวนคี่

เลขยกกำลัง (Exponents): $a^n$

สมบัติสำคัญ

  • $a^m \cdot a^n = a^{m+n}$
  • $\frac{a^m}{a^n} = a^{m-n}$
  • $(a^m)^n = a^{mn}$
  • $(ab)^n = a^n b^n$
  • $\left(\frac{a}{b}\right)^n = \frac{a^n}{b^n}$
  • $a^0 = 1$ ($a \neq 0$)
  • $a^{-n} = \frac{1}{a^n}$ ($a \neq 0$)
  • $a^{\frac{m}{n}} = \sqrt[n]{a^m}$

1.1.5 ตัวอย่างโจทย์และแนวทางการแก้ปัญหา

การทำความเข้าใจเรื่องจำนวนจริงและสมบัติจะช่วยให้แก้ปัญหาคณิตศาสตร์พื้นฐานได้อย่างแม่นยำ รวมถึงเป็นพื้นฐานสำคัญสำหรับหัวข้อที่ซับซ้อนขึ้นในวิทยาการคำนวณ เช่น การวิเคราะห์โดเมน/เรนจ์ของฟังก์ชัน, การจัดการข้อมูลเชิงตัวเลข, และการวิเคราะห์เงื่อนไขต่างๆ ในโปรแกรม

ตัวอย่างที่ 1: การจำแนกประเภทจำนวน

โจทย์

จงจำแนกจำนวนต่อไปนี้ว่าเป็นจำนวนชนิดใด: $\sqrt{9}$, $-3.14$, $\frac{22}{7}$, $\sqrt{5}$, $0$, $0.666\ldots$, $\frac{\pi}{2}$, $100$

แนวคิด

พิจารณาจากนิยามของจำนวนแต่ละประเภท

  • $\sqrt{9} = 3$
  • $0.666\ldots = \frac{2}{3}$

เฉลย

  • จำนวนนับ ($\mathbb{N}$): $100$
  • จำนวนเต็ม ($\mathbb{Z}$): $\sqrt{9}$ ($=3$), $0$, $100$
  • จำนวนตรรกยะ ($\mathbb{Q}$): $\sqrt{9}$ ($=3$), $-3.14$, $\frac{22}{7}$, $0$, $0.666\ldots$, $100$
  • จำนวนอตรรกยะ ($\mathbb{Q}'$): $\sqrt{5}$, $\frac{\pi}{2}$
  • จำนวนจริง ($\mathbb{R}$): ทุกจำนวนที่กำหนดให้

ตัวอย่างที่ 2: การใช้สมบัติค่าสัมบูรณ์

โจทย์

จงหาค่าของ $|2 - \sqrt{7}| + |1 - \sqrt{7}|$

แนวคิด

  1. ประเมินค่าของ $\sqrt{7}$ (ประมาณ $2.64$)
  2. พิจารณาเครื่องหมายของนิพจน์ภายในค่าสัมบูรณ์แต่ละตัว
  3. $2 - \sqrt{7}: 2 - 2.64 = -0.64$ (ติดลบ)
  4. $1 - \sqrt{7}: 1 - 2.64 = -1.64$ (ติดลบ)
  5. ใช้คุณสมบัติของค่าสัมบูรณ์: $|a| = -a$ ถ้า $a < 0$

เฉลย

$|2 - \sqrt{7}| = -(2 - \sqrt{7}) = -2 + \sqrt{7}$ (เนื่องจาก $2 < \sqrt{7}$)

$|1 - \sqrt{7}| = -(1 - \sqrt{7}) = -1 + \sqrt{7}$ (เนื่องจาก $1 < \sqrt{7}$)

ดังนั้น:

$$|2 - \sqrt{7}| + |1 - \sqrt{7}| = (-2 + \sqrt{7}) + (-1 + \sqrt{7})$$

$$= -2 + \sqrt{7} - 1 + \sqrt{7} = -3 + 2\sqrt{7}$$

ตัวอย่างที่ 3: การใช้สมบัติเลขยกกำลังและกรณฑ์

โจทย์

จงหาค่าของ $\frac{(2^3 \cdot 3^2)^2}{6^3} \cdot \sqrt[3]{-125}$

แนวคิด

  1. ใช้สมบัติเลขยกกำลัง: $(ab)^n = a^n b^n$ และ $(a^m)^n = a^{mn}$
  2. แปลงฐานให้เป็นเลขเดียวกัน (ถ้าเป็นไปได้): $6^3 = (2 \cdot 3)^3 = 2^3 \cdot 3^3$
  3. หาค่าของกรณฑ์ที่สาม: $\sqrt[3]{-125}$

เฉลย

$$\frac{(2^3 \cdot 3^2)^2}{6^3} \cdot \sqrt[3]{-125}$$

$$= \frac{2^{3 \cdot 2} \cdot 3^{2 \cdot 2}}{(2 \cdot 3)^3} \cdot (-5)$$

$$= \frac{2^6 \cdot 3^4}{2^3 \cdot 3^3} \cdot (-5)$$

$$= (2^{6-3} \cdot 3^{4-3}) \cdot (-5)$$

$$= (2^3 \cdot 3^1) \cdot (-5)$$

$$= (8 \cdot 3) \cdot (-5) = 24 \cdot (-5) = -120$$

ตัวอย่างที่ 4: การแก้ปัญหาผสมผสาน

โจทย์

ถ้า $x$ เป็นจำนวนเต็มบวก และ $\sqrt{256^x} = 2^{x+6}$ จงหาค่าของ $x$

แนวคิด

  1. แปลงฐานของเลขยกกำลังให้เป็นฐานเดียวกัน (ในที่นี้คือฐาน $2$)
  2. $256 = 2^8$
  3. ใช้สมบัติเลขยกกำลังและกรณฑ์: $\sqrt{a^m} = a^{\frac{m}{2}}$
  4. แก้สมการเลขยกกำลัง

เฉลย

$$\sqrt{256^x} = 2^{x+6}$$

$$\sqrt{(2^8)^x} = 2^{x+6}$$

$$\sqrt{2^{8x}} = 2^{x+6}$$

$$2^{\frac{8x}{2}} = 2^{x+6}$$

$$2^{4x} = 2^{x+6}$$

เนื่องจากฐานเท่ากัน เลขชี้กำลังจึงต้องเท่ากัน:

$$4x = x + 6$$

$$4x - x = 6$$

$$3x = 6$$

$$x = 2$$
