บทที่ 4.4: สมการและอสมการพหุนามตัวแปรเดียว (ดีกรีไม่เกินสี่)

ในบทเรียนนี้ เราจะขยายความเข้าใจจากการแก้สมการกำลังสองไปสู่ "สมการและอสมการพหุนามตัวแปรเดียว" ที่มีดีกรีสูงขึ้น (ไม่เกินสี่) หัวข้อนี้อาศัยความรู้เรื่องการแยกตัวประกอบพหุนามและทฤษฎีบทต่าง ๆ ที่เกี่ยวข้อง ซึ่งเป็นสิ่งจำเป็นสำหรับการแก้ปัญหาที่ซับซ้อนในคณิตศาสตร์และวิทยาการคำนวณครับ

1. สมการพหุนามตัวแปรเดียว (Polynomial Equations in One Variable)

1.1 นิยาม

สมการพหุนามตัวแปรเดียว คือ สมการที่อยู่ในรูป $P(x) = 0$ เมื่อ $P(x)$ เป็นพหุนามของตัวแปร $x$

$$a_nx^n + a_{n-1}x^{n-1} + \cdots + a_1x + a_0 = 0$$

โดยที่เราจะเน้นกรณีที่ $n$ (ดีกรี) มีค่าไม่เกิน 4

1.2 วิธีการแก้สมการพหุนาม

การแยกตัวประกอบ (Factoring)

  • การดึงตัวร่วม: ดึงพจน์ที่เหมือนกันออกไป
  • การใช้สูตรผลต่างกำลังสอง/กำลังสาม:
    $a^2 - b^2 = (a-b)(a+b)$
    $a^3 - b^3 = (a-b)(a^2 + ab + b^2)$
    $a^3 + b^3 = (a+b)(a^2 - ab + b^2)$
  • การจัดหมู่: จัดกลุ่มพจน์เพื่อดึงตัวร่วม
  • การแยกตัวประกอบดีกรีสอง: เช่น $x^2 + bx + c = (x+p)(x+q)$

ทฤษฎีบทตัวประกอบ (Factor Theorem)

ถ้า $P(c) = 0$ แล้ว $(x-c)$ เป็นตัวประกอบของ $P(x)$

ทฤษฎีบทรากตรรกยะ (Rational Root Theorem)

ถ้า $P(x) = a_nx^n + \cdots + a_0 = 0$ มีรากตรรกยะเป็น $\frac{p}{q}$ (เมื่อ $p,q$ เป็นจำนวนเต็ม, $q \neq 0$, และ ห.ร.ม. ของ $p,q$ คือ 1) แล้ว $p$ ต้องหาร $a_0$ ลงตัว และ $q$ ต้องหาร $a_n$ ลงตัว

การใช้งาน: ใช้ทฤษฎีนี้ในการ "เดา" รากที่เป็นไปได้ เพื่อใช้ทฤษฎีบทตัวประกอบต่อไป

การหารสังเคราะห์ (Synthetic Division)

เมื่อเราพบราก $c$ (จากทฤษฎีบทรากตรรกยะหรือการสังเกต) ซึ่งทำให้ $(x-c)$ เป็นตัวประกอบของ $P(x)$ เราสามารถใช้การหารสังเคราะห์เพื่อหาผลหาร $Q(x)$ โดยที่ $P(x) = (x-c)Q(x)$

ถ้า $Q(x)$ เป็นพหุนามดีกรี 2 เราสามารถใช้สูตรกำลังสองหรือการแยกตัวประกอบต่อไปได้

1.3 จำนวนรากของสมการพหุนาม

สมการพหุนามดีกรี $n$ จะมีรากไม่เกิน $n$ ราก (นับรวมรากซ้ำและรากเชิงซ้อน)

2. อสมการพหุนามตัวแปรเดียว (Polynomial Inequalities in One Variable)

2.1 นิยาม

อสมการพหุนามตัวแปรเดียว คือ อสมการที่อยู่ในรูป $P(x) < 0$, $P(x) > 0$, $P(x) \leq 0$, หรือ $P(x) \geq 0$

2.2 วิธีการแก้อสมการพหุนาม

ขั้นตอนที่ 1: จัดรูปอสมการ

ทำให้อสมการข้างหนึ่งเป็นศูนย์ (เช่น $P(x) > 0$)

ขั้นตอนที่ 2: แยกตัวประกอบ

แยกตัวประกอบของพหุนาม $P(x)$ ให้อยู่ในรูปผลคูณของตัวประกอบเชิงเส้นหรือตัวประกอบกำลังสองที่ไม่สามารถแยกได้อีก

ขั้นตอนที่ 3: หาราก (Critical Points)

หาค่า $x$ ที่ทำให้แต่ละตัวประกอบเท่ากับศูนย์ (รากของสมการ $P(x) = 0$) ค่าเหล่านี้เรียกว่าจุดวิกฤต

ขั้นตอนที่ 4: เขียนเส้นจำนวน

ลงจุดวิกฤตบนเส้นจำนวน ทำให้เส้นจำนวนแบ่งออกเป็นช่วงๆ

ขั้นตอนที่ 5: ทดสอบค่าในแต่ละช่วง

เลือกค่า $x$ หนึ่งค่าในแต่ละช่วง แล้วแทนลงใน $P(x)$ เพื่อดูว่า $P(x)$ มีเครื่องหมายเป็นบวกหรือลบในช่วงนั้น

หรือใช้กฎของเครื่องหมาย:
  • เริ่มต้นที่ช่วงขวาสุดด้วยเครื่องหมายบวกหรือลบตามสัมประสิทธิ์นำของ $P(x)$
  • สลับเครื่องหมายเมื่อผ่านรากที่มีเลขชี้กำลังคี่ (รากเดี่ยว)
  • คงเครื่องหมายเดิมเมื่อผ่านรากที่มีเลขชี้กำลังคู่ (รากซ้ำ)

ขั้นตอนที่ 6: ระบุช่วงคำตอบ

เลือกช่วงที่สอดคล้องกับเครื่องหมายอสมการที่ต้องการ:

  • $P(x) > 0$: เลือกช่วงที่ $P(x)$ เป็นบวก (ไม่รวมจุดวิกฤต)
  • $P(x) < 0$: เลือกช่วงที่ $P(x)$ เป็นลบ (ไม่รวมจุดวิกฤต)
  • $P(x) \geq 0$: เลือกช่วงที่ $P(x)$ เป็นบวก และรวมจุดวิกฤต
  • $P(x) \leq 0$: เลือกช่วงที่ $P(x)$ เป็นลบ และรวมจุดวิกฤต

3. ตัวอย่างโจทย์และแนวทางการทำข้อสอบ

ตัวอย่างที่ 1: การแก้สมการพหุนามดีกรี 3 (ดึงตัวร่วม)

คำถาม: จงหาคำตอบของสมการ $x^3 - 4x = 0$

แนวคิด:

  • ดึงตัวร่วม $x$ ออกมา
  • แยกตัวประกอบที่เหลือ (ผลต่างกำลังสอง)
  • หาค่า $x$ ที่ทำให้แต่ละตัวประกอบเป็นศูนย์

เฉลย:

$x^3 - 4x = 0$

ดึงตัวร่วม $x$: $x(x^2 - 4) = 0$

แยกตัวประกอบผลต่างกำลังสอง: $x(x-2)(x+2) = 0$

กำหนดให้แต่ละตัวประกอบเท่ากับศูนย์:

  • $x = 0$
  • $x - 2 = 0 \Rightarrow x = 2$
  • $x + 2 = 0 \Rightarrow x = -2$

คำตอบ: $x = 0, 2, -2$

ตัวอย่างที่ 2: การแก้สมการพหุนามดีกรี 3 (ทฤษฎีบทตัวประกอบและการหารสังเคราะห์)

คำถาม: จงหาคำตอบของสมการ $x^3 - 2x^2 - 5x + 6 = 0$

แนวคิด:

  • ใช้ทฤษฎีบทรากตรรกยะเพื่อเดาค่า $x$ ที่เป็นไปได้ (ตัวหารของ 6 คือ $\pm1, \pm2, \pm3, \pm6$)
  • ลองแทนค่า $x$ ที่ง่ายที่สุด เช่น $x = 1, -1, 2, -2$ เพื่อหาตัวประกอบแรก
  • เมื่อพบตัวประกอบ $(x-c)$ ให้ใช้การหารสังเคราะห์เพื่อลดดีกรีของพหุนาม
  • แยกตัวประกอบของพหุนามดีกรี 2 ที่ได้

เฉลย:

ให้ $P(x) = x^3 - 2x^2 - 5x + 6$

ลองแทน $x = 1$: $P(1) = (1)^3 - 2(1)^2 - 5(1) + 6 = 1 - 2 - 5 + 6 = 0$

ดังนั้น $(x-1)$ เป็นตัวประกอบหนึ่ง

ใช้การหารสังเคราะห์ด้วย 1:

1 |1-2-56
1-1-6
────────────────
1-1-60

ผลหารคือ $x^2 - x - 6$

ดังนั้น $P(x) = (x-1)(x^2 - x - 6) = 0$

แยกตัวประกอบของ $x^2 - x - 6$: $(x-1)(x-3)(x+2) = 0$

กำหนดให้แต่ละตัวประกอบเท่ากับศูนย์:

  • $x - 1 = 0 \Rightarrow x = 1$
  • $x - 3 = 0 \Rightarrow x = 3$
  • $x + 2 = 0 \Rightarrow x = -2$

คำตอบ: $x = 1, 3, -2$

ตัวอย่างที่ 3: การแก้อสมการพหุนามดีกรี 3

คำถาม: จงแก้อสมการ $x^3 - x^2 - 6x < 0$

เฉลย:

$x^3 - x^2 - 6x < 0$

ดึงตัวร่วม $x$: $x(x^2 - x - 6) < 0$

แยกตัวประกอบของ $x^2 - x - 6$: $x(x-3)(x+2) < 0$

หาราก (จุดวิกฤต):

  • $x = 0$
  • $x - 3 = 0 \Rightarrow x = 3$
  • $x + 2 = 0 \Rightarrow x = -2$

จุดวิกฤตคือ $-2, 0, 3$

ทดสอบช่วง:

  • ช่วง $x > 3$ (เช่น $x = 4$): $4(4-3)(4+2) = 4(1)(6) = 24 > 0$ (บวก)
  • ช่วง $0 < x < 3$ (เช่น $x = 1$): $1(1-3)(1+2) = 1(-2)(3) = -6 < 0$ (ลบ)
  • ช่วง $-2 < x < 0$ (เช่น $x = -1$): $(-1)(-1-3)(-1+2) = (-1)(-4)(1) = 4 > 0$ (บวก)
  • ช่วง $x < -2$ (เช่น $x = -3$): $(-3)(-3-3)(-3+2) = (-3)(-6)(-1) = -18 < 0$ (ลบ)

เราต้องการช่วงที่ $x(x-3)(x+2) < 0$ (น้อยกว่าศูนย์ หรือติดลบ)

คำตอบ: $(-\infty, -2) \cup (0, 3)$

4. ตัวอย่างเพิ่มเติม

ตัวอย่างที่ 4: การแก้อสมการพหุนามดีกรี 4

คำถาม: จงแก้อสมการ $(x^2 - 1)(x^2 - 4) \geq 0$

เฉลย:

$(x^2 - 1)(x^2 - 4) \geq 0$

แยกตัวประกอบ: $(x-1)(x+1)(x-2)(x+2) \geq 0$

หาราก (จุดวิกฤต): $x = -2, -1, 1, 2$

ทดสอบช่วงและพบว่าช่วงที่ $P(x) \geq 0$ คือ:

คำตอบ: $(-\infty, -2] \cup [-1, 1] \cup [2, \infty)$

ตัวอย่างที่ 5: การแก้สมการพหุนามดีกรี 4 (การจัดรูป)

คำถาม: จงหาคำตอบของสมการ $x^4 - 5x^2 + 4 = 0$

เฉลย:

ให้ $u = x^2$

สมการจะกลายเป็น: $u^2 - 5u + 4 = 0$

แยกตัวประกอบ: $(u-1)(u-4) = 0$

ดังนั้น $u = 1$ หรือ $u = 4$

แทนค่า $u$ กลับด้วย $x^2$:

  • กรณีที่ 1: $x^2 = 1 \Rightarrow x = \pm 1$
  • กรณีที่ 2: $x^2 = 4 \Rightarrow x = \pm 2$

คำตอบ: $x = -2, -1, 1, 2$

แบบฝึกหัด

ข้อ 1

จงหาคำตอบของสมการ $x^3 + 3x^2 - x - 3 = 0$

เฉลย:

จัดกลุ่ม: $(x^3 + 3x^2) - (x + 3) = 0$

$x^2(x + 3) - 1(x + 3) = 0$

$(x + 3)(x^2 - 1) = 0$

$(x + 3)(x - 1)(x + 1) = 0$

ตอบ: $x = -3, -1, 1$

ข้อ 2

จงหาคำตอบของสมการ $x^4 - 2x^3 - 7x^2 + 8x + 12 = 0$

เฉลย:

ใช้ทฤษฎีบทรากตรรกยะ ลองแทน $x = -1$:

$P(-1) = 1 + 2 - 7 - 8 + 12 = 0$

ใช้การหารสังเคราะห์และแยกตัวประกอบต่อ:

$P(x) = (x + 1)(x - 3)(x - 2)(x + 2) = 0$

ตอบ: $x = -2, -1, 2, 3$

ข้อ 3

จงแก้อสมการ $x^3 + 2x^2 - 3x \geq 0$

เฉลย:

ดึงตัวร่วม: $x(x^2 + 2x - 3) \geq 0$

แยกตัวประกอบ: $x(x + 3)(x - 1) \geq 0$

จุดวิกฤต: $x = -3, 0, 1$

ทดสอบช่วงและพบว่า:

ตอบ: $[-3, 0] \cup [1, \infty)$

ข้อ 4

จงแก้อสมการ $(x^2 - 9)(x^2 - 16) < 0$

เฉลย:

แยกตัวประกอบ: $(x - 3)(x + 3)(x - 4)(x + 4) < 0$

จุดวิกฤต: $x = -4, -3, 3, 4$

ทดสอบช่วงและพบว่าช่วงที่ติดลบคือ:

ตอบ: $(-4, -3) \cup (3, 4)$

ข้อ 5

จงหาคำตอบของสมการ $x^4 - 10x^2 + 9 = 0$

เฉลย:

ให้ $u = x^2$

$u^2 - 10u + 9 = 0$

$(u - 1)(u - 9) = 0$

$u = 1$ หรือ $u = 9$

$x^2 = 1 \Rightarrow x = \pm 1$

$x^2 = 9 \Rightarrow x = \pm 3$

ตอบ: $x = -3, -1, 1, 3$

ข้อ 6

จงหาคำตอบของสมการ $2x^3 - 18x = 0$

เฉลย:

ดึงตัวร่วม $2x$: $2x(x^2 - 9) = 0$

แยกตัวประกอบ: $2x(x - 3)(x + 3) = 0$

$2x = 0 \Rightarrow x = 0$

$x - 3 = 0 \Rightarrow x = 3$

$x + 3 = 0 \Rightarrow x = -3$

ตอบ: $x = -3, 0, 3$

ข้อ 7

จงแก้อสมการ $x^4 - 5x^2 + 4 \leq 0$

เฉลย:

แยกตัวประกอบ: $(x^2 - 1)(x^2 - 4) \leq 0$

$(x - 1)(x + 1)(x - 2)(x + 2) \leq 0$

จุดวิกฤต: $x = -2, -1, 1, 2$

ทดสอบช่วงและพบว่าช่วงที่ติดลบหรือเท่ากับศูนย์คือ:

ตอบ: $[-2, -1] \cup [1, 2]$

ข้อ 8

จงหาคำตอบของสมการ $x^4 + 3x^3 - 4x^2 = 0$

เฉลย:

ดึงตัวร่วม $x^2$: $x^2(x^2 + 3x - 4) = 0$

แยกตัวประกอบ: $x^2(x + 4)(x - 1) = 0$

$x^2 = 0 \Rightarrow x = 0$ (รากซ้ำ)

$x + 4 = 0 \Rightarrow x = -4$

$x - 1 = 0 \Rightarrow x = 1$

ตอบ: $x = -4, 0, 1$

ข้อ 9

จงแก้อสมการ $x^2(x - 2)(x + 3) > 0$

เฉลย:

จุดวิกฤต: $x = -3, 0, 2$

($x = 0$ เป็นรากซ้ำ จึงไม่สลับเครื่องหมาย)

ทดสอบช่วง:

  • $x > 2$: บวก
  • $0 < x < 2$: ลบ
  • $-3 < x < 0$: ลบ
  • $x < -3$: บวก

ตอบ: $(-\infty, -3) \cup (2, \infty)$

ข้อ 10

จงหาคำตอบทั้งหมดของสมการ $x^4 + x^3 - 7x^2 - x + 6 = 0$

เฉลย:

ใช้ทฤษฎีบทรากตรรกยะ ลองแทน $x = 1$:

$P(1) = 1 + 1 - 7 - 1 + 6 = 0$

ใช้การหารสังเคราะห์ต่อเนื่อง:

$P(x) = (x - 1)(x + 1)(x + 3)(x - 2) = 0$

ตอบ: $x = -3, -1, 1, 2$
