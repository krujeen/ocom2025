บทที่ 8.6: จำนวนฟิโบนักชี (Fibonacci Numbers)

ทำความเข้าใจเกี่ยวกับ "จำนวนฟิโบนักชี" ซึ่งเป็นลำดับของจำนวนที่มีคุณสมบัติพิเศษที่พจน์ถัดไปเกิดจากการบวกกันของสองพจน์ก่อนหน้า ลำดับนี้ไม่ได้ปรากฏแค่ในคณิตศาสตร์เท่านั้น แต่ยังพบเห็นได้ในธรรมชาติ ศิลปะ และวิทยาการคอมพิวเตอร์ โดยเฉพาะอย่างยิ่งในการวิเคราะห์อัลกอริทึมและการเขียนโปรแกรมเชิงเรียกซ้ำ (Recursion)

8.6.1 ความหมายและนิยาม (Definition)

นิยาม

จำนวนฟิโบนักชี (Fibonacci Numbers) คือ ลำดับของจำนวนเต็มที่เริ่มต้นด้วย 0 และ 1 (หรือ 1 และ 1) โดยที่พจน์ถัดไปในลำดับคือผลบวกของสองพจน์ก่อนหน้า

รูปแบบลำดับ

ส่วนใหญ่นิยมกำหนดให้: $F_0 = 0, F_1 = 1, F_2 = 1, F_3 = 2, F_4 = 3, F_5 = 5, F_6 = 8, F_7 = 13, \ldots$

8.6.2 ความสัมพันธ์เวียนเกิด (Recurrence Relation)

ความสัมพันธ์เวียนเกิด

นิยามของจำนวนฟิโบนักชีสามารถเขียนในรูปความสัมพันธ์เวียนเกิด (recurrence relation) ได้ดังนี้:

$$F_n = F_{n-1} + F_{n-2}$$

เมื่อ $n \geq 2$

เงื่อนไขเริ่มต้น (Initial Conditions)

การกำหนดเงื่อนไขเริ่มต้นมีความสำคัญเพื่อให้ลำดับมีความชัดเจน โดยมีสองรูปแบบที่ใช้กันบ่อย:

แบบที่ 1 (นิยมใช้ใน Computer Science)

  • $F_0 = 0$
  • $F_1 = 1$
  • ลำดับ: $0, 1, 1, 2, 3, 5, 8, 13, \ldots$

แบบที่ 2 (นิยมใช้ใน Mathematics)

  • $F_1 = 1$
  • $F_2 = 1$
  • ลำดับ: $1, 1, 2, 3, 5, 8, 13, \ldots$

หมายเหตุ: ในเอกสารนี้ เราจะใช้รูปแบบที่ 1 ($F_0 = 0, F_1 = 1$) เป็นหลักในการทำโจทย์เว้นแต่จะระบุเป็นอย่างอื่น

8.6.3 คุณสมบัติและการประยุกต์ (Properties and Applications)

ก. ปรากฏในธรรมชาติ

จำนวนฟิโบนักชีพบได้ในรูปแบบการจัดเรียงของกลีบดอกไม้, เกลียวของเปลือกหอย, กิ่งก้านของต้นไม้, และเกลียวของเมล็ดทานตะวัน

ข. อัตราส่วนทองคำ (Golden Ratio)

เมื่อ $n$ มีค่ามากพอ อัตราส่วนของพจน์ที่อยู่ติดกัน $F_n / F_{n-1}$ จะมีค่าเข้าใกล้อัตราส่วนทองคำ ($\phi \approx 1.618$)

ค. วิทยาการคอมพิวเตอร์

  • อัลกอริทึมเรียกซ้ำ (Recursive Algorithms): การคำนวณจำนวนฟิโบนักชีเป็นตัวอย่างคลาสสิกของการเขียนโปรแกรมแบบเรียกซ้ำ
  • การวิเคราะห์ประสิทธิภาพของอัลกอริทึม: เช่น อัลกอริทึม Euclidean สำหรับการหา ห.ร.ม. ที่มีประสิทธิภาพแย่ที่สุดคือเมื่อใส่อินพุตเป็นจำนวนฟิโบนักชีที่เรียงติดกัน
  • โครงสร้างข้อมูล: บางโครงสร้างข้อมูลใช้หลักการของฟิโบนักชี

8.6.4 ตัวอย่างโจทย์และแนวทางการทำข้อสอบ

โจทย์ข้อที่ 1: การหาพจน์ที่ n ของจำนวนฟิโบนักชี

คำถาม: จงหาค่าของ $F_8$ (เมื่อ $F_0 = 0, F_1 = 1$)

แนวคิด

  • ไล่ลำดับจำนวนฟิโบนักชีไปเรื่อย ๆ จนถึงพจน์ที่ต้องการ โดยใช้ความสัมพันธ์เวียนเกิด $F_n = F_{n-1} + F_{n-2}$

เฉลย

$F_0 = 0$

$F_1 = 1$

$F_2 = F_1 + F_0 = 1 + 0 = 1$

$F_3 = F_2 + F_1 = 1 + 1 = 2$

$F_4 = F_3 + F_2 = 2 + 1 = 3$

$F_5 = F_4 + F_3 = 3 + 2 = 5$

$F_6 = F_5 + F_4 = 5 + 3 = 8$

$F_7 = F_6 + F_5 = 8 + 5 = 13$

$F_8 = F_7 + F_6 = 13 + 8 = 21$

คำตอบ: 21

โจทย์ข้อที่ 2: การประยุกต์ใช้จำนวนฟิโบนักชี (การนับวิธีขึ้นบันได)

คำถาม: มีบันได 10 ขั้น ถ้าคนสามารถก้าวขึ้นบันไดได้ครั้งละ 1 ขั้น หรือ 2 ขั้นเท่านั้น จงหาวิธีขึ้นบันไดทั้งหมดกี่วิธี

แนวคิด

  • ปัญหานี้เป็นปัญหาคลาสสิกที่สามารถเชื่อมโยงกับจำนวนฟิโบนักชีได้
  • ให้ $W_n$ เป็นจำนวนวิธีในการขึ้นบันได $n$ ขั้น
  • $W_1 = 1$ (1 ขั้น)
  • $W_2 = 2$ (1+1, 2)
  • $W_n = W_{n-1} + W_{n-2}$ (ถ้าก้าวสุดท้ายเป็น 1 ขั้น จะเหลือ $W_{n-1}$ วิธี; ถ้าก้าวสุดท้ายเป็น 2 ขั้น จะเหลือ $W_{n-2}$ วิธี)
  • สังเกตความสัมพันธ์กับจำนวนฟิโบนักชี

เฉลย

ให้ $W_n$ แทนจำนวนวิธีในการขึ้นบันได $n$ ขั้น

$W_0 = 1$ (เริ่มต้นที่ขั้น 0, มี 1 วิธี คือไม่ต้องก้าว) - เพื่อให้สอดคล้องกับ $F_n$

$W_1 = 1$ (ก้าว 1 ขั้น)

$W_2 = W_1 + W_0 = 1 + 1 = 2$ (ก้าว 1+1 หรือ 2)

$W_3 = W_2 + W_1 = 2 + 1 = 3$

$W_4 = W_3 + W_2 = 3 + 2 = 5$

$W_5 = W_4 + W_3 = 5 + 3 = 8$

$W_6 = W_5 + W_4 = 8 + 5 = 13$

$W_7 = W_6 + W_5 = 13 + 8 = 21$

$W_8 = W_7 + W_6 = 21 + 13 = 34$

$W_9 = W_8 + W_7 = 34 + 21 = 55$

$W_{10} = W_9 + W_8 = 55 + 34 = 89$

จะเห็นว่า $W_n = F_{n+1}$ (เมื่อ $F_0 = 0, F_1 = 1$)

ดังนั้น $W_{10} = F_{11} = 89$

คำตอบ: 89 วิธี

โจทย์ข้อที่ 3: การระบุพจน์ในลำดับฟิโบนักชี

คำถาม: จงพิจารณาว่าจำนวน 55 เป็นพจน์ที่เท่าไหร่ในลำดับฟิโบนักชี (เมื่อ $F_0 = 0, F_1 = 1$)

แนวคิด

  • ไล่ลำดับฟิโบนักชีไปเรื่อย ๆ จนกว่าจะเจอ 55

เฉลย

$F_0 = 0$

$F_1 = 1$

$F_2 = 1$

$F_3 = 2$

$F_4 = 3$

$F_5 = 5$

$F_6 = 8$

$F_7 = 13$

$F_8 = 21$

$F_9 = 34$

$F_{10} = 55$

คำตอบ: 55 เป็นพจน์ที่ $F_{10}$ (หรือพจน์ที่ 11 ถ้าเริ่มนับจาก $F_0$)

โจทย์ข้อที่ 4: การหาผลรวมของจำนวนฟิโบนักชี

คำถาม: จงหาผลรวมของจำนวนฟิโบนักชีตั้งแต่ $F_0$ ถึง $F_7$ (เมื่อ $F_0 = 0, F_1 = 1$)

แนวคิด

  • ผลรวมของ $F_0 + F_1 + \cdots + F_n = F_{n+2} - 1$
  • พิสูจน์ได้โดยการไล่บวก หรือใช้หลักอุปนัยเชิงคณิตศาสตร์ (Mathematical Induction)

เฉลย

ผลรวมของ $F_0 + F_1 + \cdots + F_7$

เราทราบว่า $F_0 = 0, F_1 = 1, F_2 = 1, F_3 = 2, F_4 = 3, F_5 = 5, F_6 = 8, F_7 = 13$

ผลรวม = $0 + 1 + 1 + 2 + 3 + 5 + 8 + 13 = 33$

ตรวจสอบด้วยสูตร: $F_{7+2} - 1 = F_9 - 1$

$F_9 = F_8 + F_7 = 21 + 13 = 34$

ดังนั้น $F_9 - 1 = 34 - 1 = 33$

คำตอบ: 33

โจทย์ข้อที่ 5: การประยุกต์ใช้จำนวนฟิโบนักชี (การจัดเรียงที่ไม่ติดกัน)

คำถาม: มีกระเบื้องรูปสี่เหลี่ยมจัตุรัสขนาด 1x1 จำนวน $n$ แผ่นเรียงเป็นแถวยาว จงหาวิธีการวางกระเบื้องสีดำบนกระเบื้องเหล่านี้ โดยที่กระเบื้องสีดำห้ามอยู่ติดกัน (ไม่จำเป็นต้องวางทุกแผ่น)

แนวคิด

  • ให้ $A_n$ เป็นจำนวนวิธีในการวางกระเบื้องสีดำบนกระเบื้อง $n$ แผ่น
  • พิจารณาสองกรณีสำหรับกระเบื้องแผ่นที่ $n$:
  • กระเบื้องแผ่นที่ $n$ เป็นสีขาว: จำนวนวิธีคือ $A_{n-1}$
  • กระเบื้องแผ่นที่ $n$ เป็นสีดำ: กระเบื้องแผ่นที่ $n-1$ ต้องเป็นสีขาว จำนวนวิธีคือ $A_{n-2}$
  • ดังนั้น $A_n = A_{n-1} + A_{n-2}$ (ความสัมพันธ์แบบฟิโบนักชี)

เฉลย

ให้ $A_n$ คือจำนวนวิธีในการวางกระเบื้อง

$A_0 = 1$ (ไม่วางเลย 1 วิธี)

$A_1 = 2$ (ขาว 1 วิธี, ดำ 1 วิธี)

$A_2 = A_1 + A_0 = 2 + 1 = 3$ (ขาว-ขาว, ขาว-ดำ, ดำ-ขาว)

$A_3 = A_2 + A_1 = 3 + 2 = 5$

จะเห็นว่า $A_n = F_{n+2}$ (เมื่อ $F_0 = 0, F_1 = 1$)

ตัวอย่าง: ถ้ามีกระเบื้อง $n$ แผ่น

  • $n = 1$: (W), (B) ⟹ 2 วิธี ($F_3$)
  • $n = 2$: (WW), (WB), (BW) ⟹ 3 วิธี ($F_4$)
  • $n = 3$: (WWW), (WWB), (WBW), (BWW), (BWB) ⟹ 5 วิธี ($F_5$)

คำตอบ: จำนวนวิธีคือ $F_{n+2}$ วิธี

8.6.5 ตัวอย่างข้อสอบฝึกทำ: จำนวนฟิโบนักชี (Fibonacci Numbers)

เอกสารนี้รวบรวมตัวอย่างข้อสอบสำหรับนักเรียนได้ฝึกทำในหัวข้อ "จำนวนฟิโบนักชี" เพื่อเสริมสร้างความเข้าใจและเตรียมความพร้อมสำหรับการสอบ สอวน. คอมพิวเตอร์ครับ

โจทย์ข้อที่ 1: การหาพจน์ที่ n ของจำนวนฟิโบนักชี

คำถาม: จงหาค่าของ $F_9$ (เมื่อ $F_0 = 0, F_1 = 1$)

แนวคิด

  • ไล่ลำดับจำนวนฟิโบนักชีไปเรื่อย ๆ จนถึงพจน์ที่ต้องการ โดยใช้ความสัมพันธ์เวียนเกิด $F_n = F_{n-1} + F_{n-2}$

เฉลย

$F_0 = 0$

$F_1 = 1$

$F_2 = F_1 + F_0 = 1 + 0 = 1$

$F_3 = F_2 + F_1 = 1 + 1 = 2$

$F_4 = F_3 + F_2 = 2 + 1 = 3$

$F_5 = F_4 + F_3 = 3 + 2 = 5$

$F_6 = F_5 + F_4 = 5 + 3 = 8$

$F_7 = F_6 + F_5 = 8 + 5 = 13$

$F_8 = F_7 + F_6 = 13 + 8 = 21$

$F_9 = F_8 + F_7 = 21 + 13 = 34$

คำตอบ: 34

โจทย์ข้อที่ 2: การประยุกต์ใช้จำนวนฟิโบนักชี (การนับวิธีขึ้นบันได)

คำถาม: มีบันได 8 ขั้น ถ้าคนสามารถก้าวขึ้นบันไดได้ครั้งละ 1 ขั้น หรือ 2 ขั้นเท่านั้น จงหาวิธีขึ้นบันไดทั้งหมดกี่วิธี?

แนวคิด

  • ปัญหานี้สามารถเชื่อมโยงกับจำนวนฟิโบนักชีได้ ($W_n = W_{n-1} + W_{n-2}$).
  • ถ้า $F_0 = 0, F_1 = 1$, จำนวนวิธีขึ้นบันได $n$ ขั้นคือ $F_{n+1}$

เฉลย

ให้ $W_n$ แทนจำนวนวิธีในการขึ้นบันได $n$ ขั้น

$W_0 = 1$ (1 วิธีที่จะอยู่กับที่)

$W_1 = 1$ (1 วิธี: ก้าว 1 ขั้น)

$W_2 = W_1 + W_0 = 1 + 1 = 2$ (2 วิธี: 1+1, 2)

$W_3 = W_2 + W_1 = 2 + 1 = 3$

$W_4 = W_3 + W_2 = 3 + 2 = 5$

$W_5 = W_4 + W_3 = 5 + 3 = 8$

$W_6 = W_5 + W_4 = 8 + 5 = 13$

$W_7 = W_6 + W_5 = 13 + 8 = 21$

$W_8 = W_7 + W_6 = 21 + 13 = 34$

คำตอบ: 34 วิธี

โจทย์ข้อที่ 3: การระบุพจน์ในลำดับฟิโบนักชี

คำถาม: จงพิจารณาว่าจำนวน 89 เป็นพจน์ที่เท่าไหร่ในลำดับฟิโบนักชี (เมื่อ $F_0 = 0, F_1 = 1$)

แนวคิด

  • ไล่ลำดับฟิโบนักชีไปเรื่อย ๆ จนกว่าจะเจอ 89

เฉลย

$F_0 = 0$

$F_1 = 1$

$F_2 = 1$

$F_3 = 2$

$F_4 = 3$

$F_5 = 5$

$F_6 = 8$

$F_7 = 13$

$F_8 = 21$

$F_9 = 34$

$F_{10} = 55$

$F_{11} = 89$

คำตอบ: 89 เป็นพจน์ที่ $F_{11}$ (หรือพจน์ที่ 12 ถ้าเริ่มนับจาก $F_0$)

โจทย์ข้อที่ 4: การหาผลรวมของจำนวนฟิโบนักชี

คำถาม: จงหาผลรวมของจำนวนฟิโบนักชีตั้งแต่ $F_0$ ถึง $F_6$ (เมื่อ $F_0 = 0, F_1 = 1$)

แนวคิด

  • ผลรวมของ $F_0 + F_1 + \cdots + F_n = F_{n+2} - 1$

เฉลย

ลำดับฟิโบนักชี: $F_0 = 0, F_1 = 1, F_2 = 1, F_3 = 2, F_4 = 3, F_5 = 5, F_6 = 8$

ผลรวม = $0 + 1 + 1 + 2 + 3 + 5 + 8 = 20$

ตรวจสอบด้วยสูตร: $F_{6+2} - 1 = F_8 - 1$

จากข้อ 1, $F_8 = 21$

ดังนั้น $F_8 - 1 = 21 - 1 = 20$

คำตอบ: 20

โจทย์ข้อที่ 5: การระบุจำนวนฟิโบนักชีในรูปแบบอื่น

คำถาม: ถ้า $A = F_5$ และ $B = F_6$ (เมื่อ $F_0 = 0, F_1 = 1$) จงหา $A + B$

แนวคิด

  • หาค่าของ $F_5$ และ $F_6$ ก่อน แล้วนำมาบวกกัน

เฉลย

$F_5 = 5$

$F_6 = 8$

$A + B = F_5 + F_6 = 5 + 8 = 13$

(ซึ่งก็คือ $F_7$)

คำตอบ: 13

โจทย์ข้อที่ 6: การคำนวณจำนวนฟิโบนักชีจากเงื่อนไขเริ่มต้นที่ต่างออกไป

คำถาม: กำหนดลำดับ $G_n$ โดย $G_n = G_{n-1} + G_{n-2}$ และ $G_1 = 1, G_2 = 3$ จงหา $G_5$

แนวคิด

  • ลำดับนี้เป็นลำดับที่คล้ายฟิโบนักชีแต่มีเงื่อนไขเริ่มต้นที่ต่างออกไป
  • ใช้ความสัมพันธ์เวียนเกิดและเงื่อนไขเริ่มต้นที่กำหนดให้

เฉลย

$G_1 = 1$

$G_2 = 3$

$G_3 = G_2 + G_1 = 3 + 1 = 4$

$G_4 = G_3 + G_2 = 4 + 3 = 7$

$G_5 = G_4 + G_3 = 7 + 4 = 11$

คำตอบ: 11

โจทย์ข้อที่ 7: การประยุกต์ใช้จำนวนฟิโบนักชี (การจัดเรียงที่ไม่ติดกัน)

คำถาม: มีกระเบื้อง 5 แผ่นเรียงกันเป็นแถวยาว จงหาวิธีการวางกระเบื้องสีดำบนกระเบื้องเหล่านี้ โดยที่กระเบื้องสีดำห้ามอยู่ติดกัน (ไม่จำเป็นต้องวางทุกแผ่น)

แนวคิด

  • ปัญหานี้สามารถเชื่อมโยงกับจำนวนฟิโบนักชีได้ ($A_n = A_{n-1} + A_{n-2}$).
  • ถ้า $F_0 = 0, F_1 = 1$, จำนวนวิธีคือ $F_{n+2}$

เฉลย

ให้ $A_n$ คือจำนวนวิธีในการวางกระเบื้อง $n$ แผ่น โดยไม่มีกระเบื้องสีดำติดกัน

$A_0 = 1$ (ไม่วางเลยบน 0 แผ่น)

$A_1 = 2$ (วาง W หรือ B)

$A_2 = 3$ (WW, WB, BW)

$A_3 = 5$ (WWW, WWB, WBW, BWW, BWB)

$A_4 = 8$

$A_5 = 13$

จะเห็นว่า $A_n = F_{n+2}$ (เมื่อ $F_0 = 0, F_1 = 1$)

ดังนั้น $A_5 = F_{5+2} = F_7 = 13$

คำตอบ: 13 วิธี

โจทย์ข้อที่ 8: การหาสมาชิกของลำดับฟิโบนักชีที่กำหนดเงื่อนไข

คำถาม: ถ้า $F_n = 144$ (เมื่อ $F_0 = 0, F_1 = 1$) จงหาค่าของ $n$

แนวคิด

  • ไล่ลำดับฟิโบนักชีไปเรื่อยๆ จนเจอ 144 และระบุตำแหน่ง

เฉลย

$F_0 = 0$

$F_1 = 1$

$F_2 = 1$

$F_3 = 2$

$F_4 = 3$

$F_5 = 5$

$F_6 = 8$

$F_7 = 13$

$F_8 = 21$

$F_9 = 34$

$F_{10} = 55$

$F_{11} = 89$

$F_{12} = 144$

คำตอบ: $n = 12$

โจทย์ข้อที่ 9: การหาพจน์ในลำดับฟิโบนักชีจากผลบวก

คำถาม: ถ้าผลรวมของจำนวนฟิโบนักชีตั้งแต่ $F_0$ ถึง $F_n$ เท่ากับ 20 (เมื่อ $F_0 = 0, F_1 = 1$) จงหาค่า $n$

แนวคิด

  • ใช้สูตรผลรวม: $F_0 + F_1 + \cdots + F_n = F_{n+2} - 1$
  • แก้สมการหา $F_{n+2}$ แล้วระบุ $n$

เฉลย

ผลรวม = 20

จากสูตร $F_{n+2} - 1 = 20$

$F_{n+2} = 21$

เราทราบว่า $F_8 = 21$

ดังนั้น $n + 2 = 8$

$n = 6$

คำตอบ: $n = 6$

โจทย์ข้อที่ 10: การประยุกต์ใช้จำนวนฟิโบนักชีในชีวิตประจำวัน

คำถาม: การเจริญเติบโตของจำนวนคู่กระต่าย (ซึ่งสืบพันธุ์เมื่ออายุ 1 เดือน และให้ลูก 1 คู่ทุกเดือน) สามารถจำลองด้วยจำนวนฟิโบนักชี ถ้าเริ่มต้นมีกระต่ายวัยเด็ก 1 คู่ ($F_1$) เมื่อสิ้นสุดเดือนที่ 5 จะมีกระต่ายทั้งหมดกี่คู่?

แนวคิด

  • โจทย์นี้เป็นต้นกำเนิดของลำดับฟิโบนักชี
  • $F_1 = 1$ (คู่เริ่มต้น)
  • สิ้นสุดเดือนที่ 1: กระต่ายวัยเด็ก 1 คู่ (ยังไม่สืบพันธุ์)
  • สิ้นสุดเดือนที่ 2: กระต่ายวัยผู้ใหญ่ 1 คู่ (เริ่มสืบพันธุ์)
  • จำนวนคู่กระต่ายที่สิ้นสุดเดือนที่ $k$ จะเป็น $F_{k+1}$ (เมื่อ $F_1 = 1, F_2 = 1$ แบบที่ 2) หรือ $F_{k+2}$ (เมื่อ $F_0 = 0, F_1 = 1$ แบบที่ 1)
  • โดยทั่วไป ถ้าเริ่มต้นมี 1 คู่ที่ยังไม่พร้อมสืบพันธุ์ $F_n$ จะแทนจำนวนคู่กระต่ายเมื่อสิ้นสุดเดือนที่ $n-1$

เฉลย

ลำดับฟิโบนักชี ($F_n$ เมื่อ $F_0 = 0, F_1 = 1$) แสดงจำนวนคู่กระต่ายเมื่อสิ้นสุดเดือนที่ $n-1$

เดือนที่ 0 (เริ่มต้น): $F_1 = 1$ คู่ (เด็ก)

เดือนที่ 1 (สิ้นสุด): $F_2 = 1$ คู่ (ผู้ใหญ่)

เดือนที่ 2 (สิ้นสุด): $F_3 = 2$ คู่ (ผู้ใหญ่ 1 + ลูก 1)

เดือนที่ 3 (สิ้นสุด): $F_4 = 3$ คู่

เดือนที่ 4 (สิ้นสุด): $F_5 = 5$ คู่

เดือนที่ 5 (สิ้นสุด): $F_6 = 8$ คู่

ดังนั้น เมื่อสิ้นสุดเดือนที่ 5 จะมีกระต่าย 8 คู่

คำตอบ: 8 คู่
