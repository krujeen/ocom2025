บทที่ 3.3: ประเภทของฟังก์ชัน

ศึกษาการดำเนินการต่าง ๆ ที่สามารถทำได้กับฟังก์ชัน ไม่ว่าจะเป็นการบวก ลบ คูณ หาร รวมถึงฟังก์ชันที่ซ้อนกัน และฟังก์ชันผกผัน ซึ่งเป็นพื้นฐานสำคัญในการแก้ปัญหาทางคณิตศาสตร์และวิทยาการคอมพิวเตอร์

3.3.1 การดำเนินการของฟังก์ชัน (Algebra of Functions)

ให้ $f$ และ $g$ เป็นฟังก์ชันที่มีโดเมนเป็น $D_f$ และ $D_g$ ตามลำดับ การดำเนินการทางพีชคณิตระหว่างฟังก์ชันสามารถทำได้ดังนี้ โดยโดเมนของฟังก์ชันใหม่ที่เกิดขึ้นจะเท่ากับ $D_f \cap D_g$ (ยกเว้นการหาร)

ก. การบวกฟังก์ชัน (Sum of Functions)

$$(f+g)(x) = f(x) + g(x)$$ $$\text{โดเมน: } D_{f+g} = D_f \cap D_g$$

ข. การลบฟังก์ชัน (Difference of Functions)

$$(f-g)(x) = f(x) - g(x)$$ $$\text{โดเมน: } D_{f-g} = D_f \cap D_g$$

ค. การคูณฟังก์ชัน (Product of Functions)

$$(fg)(x) = f(x) \cdot g(x)$$ $$\text{โดเมน: } D_{fg} = D_f \cap D_g$$

ง. การหารฟังก์ชัน (Quotient of Functions)

$$\left(\frac{f}{g}\right)(x) = \frac{f(x)}{g(x)}$$ $$\text{โดเมน: } D_{f/g} = \{x \mid x \in D_f \cap D_g \text{ และ } g(x) \neq 0\}$$

ตัวอย่าง การดำเนินการของฟังก์ชัน

ให้ $f(x) = x^2 - 1$ และ $g(x) = x + 3$

  • $(f+g)(x) = (x^2 - 1) + (x + 3) = x^2 + x + 2$
  • $(f-g)(x) = (x^2 - 1) - (x + 3) = x^2 - x - 4$
  • $(fg)(x) = (x^2 - 1)(x + 3) = x^3 + 3x^2 - x - 3$
  • $\left(\frac{f}{g}\right)(x) = \frac{x^2 - 1}{x + 3}$ โดยที่ $x \neq -3$

3.3.2 ฟังก์ชันประกอบ (Composite Functions)

ฟังก์ชันประกอบคือการนำฟังก์ชันสองฟังก์ชันหรือมากกว่ามาทำงานต่อเนื่องกัน โดยผลลัพธ์ของฟังก์ชันแรกจะกลายเป็นอินพุตของฟังก์ชันถัดไป

ก. นิยามของฟังก์ชันประกอบ

กำหนดฟังก์ชัน $f$ และ $g$

$(f \circ g)(x)$ (อ่านว่า "เอฟ โอ จี ของ x" หรือ "เอฟ คอมโพสิท จี ของ x")

หมายถึง $f(g(x))$

ฟังก์ชัน $g$ ทำงานก่อน แล้วนำผลลัพธ์จาก $g(x)$ ไปเป็นอินพุตให้กับฟังก์ชัน $f$

$(g \circ f)(x)$ (อ่านว่า "จี โอ เอฟ ของ x" หรือ "จี คอมโพสิท เอฟ ของ x")

หมายถึง $g(f(x))$

ฟังก์ชัน $f$ ทำงานก่อน แล้วนำผลลัพธ์จาก $f(x)$ ไปเป็นอินพุตให้กับฟังก์ชัน $g$

ข. โดเมนของฟังก์ชันประกอบ

$$D_{f \circ g} = \{x \mid x \in D_g \text{ และ } g(x) \in D_f\}$$

เงื่อนไขคือ $x$ ต้องอยู่ในโดเมนของ $g$ และผลลัพธ์ของ $g(x)$ ต้องอยู่ในโดเมนของ $f$

ตัวอย่าง ฟังก์ชันประกอบ

ให้ $f(x) = x^2 + 1$ และ $g(x) = 2x - 3$

  • $(f \circ g)(x) = f(g(x)) = f(2x - 3) = (2x - 3)^2 + 1 = 4x^2 - 12x + 9 + 1 = 4x^2 - 12x + 10$
  • $(g \circ f)(x) = g(f(x)) = g(x^2 + 1) = 2(x^2 + 1) - 3 = 2x^2 + 2 - 3 = 2x^2 - 1$

สังเกต: $(f \circ g)(x) \neq (g \circ f)(x)$ โดยทั่วไป

3.3.3 ฟังก์ชันผกผัน (Inverse Functions)

ฟังก์ชันผกผันคือฟังก์ชันที่ "ย้อนกลับ" การทำงานของฟังก์ชันเดิม

ก. ฟังก์ชันหนึ่งต่อหนึ่ง (One-to-One Function หรือ Injective Function)

นิยาม: ฟังก์ชันหนึ่งต่อหนึ่ง

ฟังก์ชัน $f$ จะเป็นฟังก์ชันหนึ่งต่อหนึ่ง ก็ต่อเมื่อสมาชิกแต่ละตัวในโดเมนจับคู่กับสมาชิกในเรนจ์ที่ไม่ซ้ำกัน

กล่าวคือ ถ้า $f(x_1) = f(x_2)$ แล้ว $x_1 = x_2$ เสมอ

การทดสอบเส้นแนวนอน (Horizontal Line Test)

ใช้ตรวจสอบว่าฟังก์ชันเป็นหนึ่งต่อหนึ่งหรือไม่ โดยการลากเส้นแนวนอน (เส้นตรงขนานกับแกน X) ตัดกราฟ

ถ้าเส้นแนวนอนตัดกราฟเพียงจุดเดียวสำหรับทุกเส้นแนวนอน แสดงว่าฟังก์ชันนั้นเป็นหนึ่งต่อหนึ่ง

หมายเหตุ: ฟังก์ชันจะมีฟังก์ชันผกผันได้ก็ต่อเมื่อฟังก์ชันนั้นเป็น ฟังก์ชันหนึ่งต่อหนึ่ง

ข. การหาฟังก์ชันผกผัน $(f^{-1}(x))$

ขั้นตอนการหาฟังก์ชันผกผัน

  1. ให้ $y = f(x)$
  2. สลับตัวแปร $x$ กับ $y$ ในสมการ (ได้ $x = f(y)$)
  3. แก้สมการที่ได้ในขั้นตอนที่ 2 เพื่อหา $y$ ในรูปของ $x$
  4. แทน $y$ ด้วย $f^{-1}(x)$

ความสัมพันธ์ระหว่างโดเมนและเรนจ์ของฟังก์ชันกับฟังก์ชันผกผัน

  • โดเมนของ $f^{-1}$ คือ เรนจ์ของ $f$ ($D_{f^{-1}} = R_f$)
  • เรนจ์ของ $f^{-1}$ คือ โดเมนของ $f$ ($R_{f^{-1}} = D_f$)

ตัวอย่าง การหาฟังก์ชันผกผัน

ให้ $f(x) = 3x - 5$

ขั้นตอน:

  1. ให้ $y = 3x - 5$
  2. สลับตัวแปร: $x = 3y - 5$
  3. แก้สมการหา $y$:
    $x + 5 = 3y$
    $y = \frac{x + 5}{3}$
  4. ดังนั้น $f^{-1}(x) = \frac{x + 5}{3}$

3.3.4 ตัวอย่างโจทย์และแนวทางการทำข้อสอบ

โจทย์ข้อที่ 1: การดำเนินการของฟังก์ชัน

คำถาม: กำหนดให้ $f(x) = x^2 - 1$ และ $g(x) = x + 3$

จงหา:

  • ก. $(f+g)(x)$ และ $D_{f+g}$
  • ข. $(f-g)(x)$ และ $D_{f-g}$
  • ค. $(fg)(x)$ และ $D_{fg}$
  • ง. $\left(\frac{f}{g}\right)(x)$ และ $D_{f/g}$

แนวคิด:

  • โดเมนของ $f(x)$ และ $g(x)$ เป็นจำนวนจริงทั้งหมด ($\mathbb{R}$)
  • ใช้สูตรการดำเนินการแต่ละแบบ
  • สำหรับการหาร ต้องระบุเงื่อนไขที่ตัวส่วนไม่เป็นศูนย์

เฉลย:

โดเมนของ $f(x)$ คือ $\mathbb{R}$ ($D_f = \mathbb{R}$)

โดเมนของ $g(x)$ คือ $\mathbb{R}$ ($D_g = \mathbb{R}$)

ดังนั้น $D_f \cap D_g = \mathbb{R}$

ก. $(f+g)(x) = f(x) + g(x) = (x^2 - 1) + (x + 3) = x^2 + x + 2$

$D_{f+g} = \mathbb{R}$

ข. $(f-g)(x) = f(x) - g(x) = (x^2 - 1) - (x + 3) = x^2 - x - 4$

$D_{f-g} = \mathbb{R}$

ค. $(fg)(x) = f(x) \cdot g(x) = (x^2 - 1)(x + 3) = x^3 + 3x^2 - x - 3$

$D_{fg} = \mathbb{R}$

ง. $\left(\frac{f}{g}\right)(x) = \frac{f(x)}{g(x)} = \frac{x^2 - 1}{x + 3}$

เงื่อนไข $g(x) \neq 0$: $x + 3 \neq 0 \Rightarrow x \neq -3$

$D_{f/g} = \{x \mid x \in \mathbb{R}, x \neq -3\}$

โจทย์ข้อที่ 2: ฟังก์ชันประกอบ

คำถาม: กำหนดให้ $f(x) = \sqrt{x}$ และ $g(x) = x^2 - 4$ จงหา:

  • ก. $(f \circ g)(x)$ และ $D_{f \circ g}$
  • ข. $(g \circ f)(x)$ และ $D_{g \circ f}$

แนวคิด:

  • โดเมนของ $f(x)$: $D_f = [0, \infty)$
  • โดเมนของ $g(x)$: $D_g = \mathbb{R}$
  • ใช้สูตรฟังก์ชันประกอบ และพิจารณาเงื่อนไขโดเมน

เฉลย:

ก. $(f \circ g)(x) = f(g(x)) = f(x^2 - 4) = \sqrt{x^2 - 4}$

โดเมน $D_{f \circ g}$:

  • $x$ ต้องอยู่ใน $D_g$ (ซึ่งคือ $\mathbb{R}$)
  • $g(x)$ ต้องอยู่ใน $D_f$ (คือ $[0, \infty)$)

ดังนั้น $x^2 - 4 \geq 0$

$x^2 \geq 4$

$|x| \geq 2 \Rightarrow x \leq -2$ หรือ $x \geq 2$

$D_{f \circ g} = (-\infty, -2] \cup [2, \infty)$

ข. $(g \circ f)(x) = g(f(x)) = g(\sqrt{x}) = (\sqrt{x})^2 - 4 = x - 4$

โดเมน $D_{g \circ f}$:

  • $x$ ต้องอยู่ใน $D_f$ (คือ $[0, \infty)$)
  • $f(x)$ ต้องอยู่ใน $D_g$ (ซึ่งคือ $\mathbb{R}$)

เงื่อนไข $f(x) \in \mathbb{R}$ เป็นจริงเสมอเมื่อ $x \in D_f$

ดังนั้น $D_{g \circ f} = [0, \infty)$

โจทย์ข้อที่ 3: ฟังก์ชันผกผัน

คำถาม: กำหนดให้ $f(x) = \frac{2x + 1}{x - 3}$ จงหา $f^{-1}(x)$

แนวคิด:

  • สลับตัวแปร $x$ กับ $y$ แล้วแก้สมการหา $y$

เฉลย:

  1. ให้ $y = \frac{2x + 1}{x - 3}$
  2. สลับตัวแปร $x$ กับ $y$: $x = \frac{2y + 1}{y - 3}$
  3. แก้สมการหา $y$:
    $x(y - 3) = 2y + 1$
    $xy - 3x = 2y + 1$
    $xy - 2y = 3x + 1$
    $y(x - 2) = 3x + 1$
    $y = \frac{3x + 1}{x - 2}$
  4. ดังนั้น $f^{-1}(x) = \frac{3x + 1}{x - 2}$

3.3.5 ตัวอย่างข้อสอบฝึกทำ

ข้อที่ 1

กำหนดให้ $f(x) = x - 5$ และ $g(x) = 2x + 1$ จงหา $(f \cdot g)(x)$

แนวคิด

ใช้สูตรการคูณฟังก์ชัน: $(f \cdot g)(x) = f(x) \cdot g(x)$

เฉลย

$(f \cdot g)(x) = f(x) \cdot g(x)$

$= (x - 5)(2x + 1)$

$= x(2x) + x(1) - 5(2x) - 5(1)$

$= 2x^2 + x - 10x - 5$

$= 2x^2 - 9x - 5$

คำตอบ: $2x^2 - 9x - 5$

ข้อที่ 2

กำหนดให้ $f(x) = x^2 - 9$ และ $g(x) = \sqrt{x + 2}$ จงหาโดเมนของ $\left(\frac{f}{g}\right)(x)$

แนวคิด

หาโดเมนของ $f(x)$ และ $g(x)$ แยกกัน

โดเมนของการหารฟังก์ชันคือส่วนที่โดเมนของทั้งสองฟังก์ชันทับซ้อนกัน และตัวส่วนต้องไม่เป็นศูนย์

เฉลย

โดเมนของ $f(x) = x^2 - 9$ คือ $\mathbb{R}$ ($D_f = \mathbb{R}$)

โดเมนของ $g(x) = \sqrt{x + 2}$ คือ $x + 2 \geq 0 \Rightarrow x \geq -2$ ($D_g = [-2, \infty)$)

โดเมนของ $\left(\frac{f}{g}\right)(x)$ คือ $D_f \cap D_g$ โดยที่ $g(x) \neq 0$

$D_f \cap D_g = \mathbb{R} \cap [-2, \infty) = [-2, \infty)$

เงื่อนไข $g(x) \neq 0$:

$\sqrt{x + 2} \neq 0$

$x + 2 \neq 0$

$x \neq -2$

ดังนั้น โดเมนของ $\left(\frac{f}{g}\right)(x)$ คือ $\{x \mid x \geq -2 \text{ และ } x \neq -2\}$ ซึ่งคือ $x > -2$

คำตอบ: $(-2, \infty)$ หรือ $\{x \mid x > -2\}$

ข้อที่ 3

กำหนดให้ $f(x) = 3x + 2$ และ $g(x) = x^2 - 1$ จงหา $(g \circ f)(x)$

แนวคิด

ใช้สูตรฟังก์ชันประกอบ: $(g \circ f)(x) = g(f(x))$

เฉลย

$(g \circ f)(x) = g(f(x))$

$= g(3x + 2)$

แทน $f(x)$ เข้าไปใน $g(x)$:

$= (3x + 2)^2 - 1$

$= ((3x)^2 + 2(3x)(2) + 2^2) - 1$

$= (9x^2 + 12x + 4) - 1$

$= 9x^2 + 12x + 3$

คำตอบ: $9x^2 + 12x + 3$

ข้อที่ 4

กำหนดให้ $f(x) = 5x - 3$ และ $g(x) = x + 4$ จงหาค่าของ $(f \circ g)(1)$

แนวคิด

หาค่า $g(1)$ ก่อน แล้วนำผลลัพธ์ไปแทนใน $f(x)$

เฉลย

หา $g(1)$:

$g(1) = 1 + 4 = 5$

หา $f(g(1)) = f(5)$:

$f(5) = 5(5) - 3 = 25 - 3 = 22$

คำตอบ: $22$

ข้อที่ 5

กำหนดให้ $f(x) = \frac{x}{2} + 7$ จงหา $f^{-1}(x)$

แนวคิด

ให้ $y = f(x)$

สลับตัวแปร $x$ กับ $y$

แก้สมการที่ได้เพื่อหา $y$ ในรูปของ $x$

เฉลย

ให้ $y = \frac{x}{2} + 7$

สลับตัวแปร $x$ กับ $y$: $x = \frac{y}{2} + 7$

แก้สมการหา $y$:

$x - 7 = \frac{y}{2}$

$2(x - 7) = y$

$y = 2x - 14$

ดังนั้น $f^{-1}(x) = 2x - 14$

คำตอบ: $f^{-1}(x) = 2x - 14$

ข้อที่ 6

กำหนดให้ $f(x) = \frac{3x + 2}{x - 4}$ จงหา $f^{-1}(x)$

แนวคิด

ให้ $y = f(x)$

สลับตัวแปร $x$ กับ $y$

แก้สมการที่ได้เพื่อหา $y$ ในรูปของ $x$

เฉลย

ให้ $y = \frac{3x + 2}{x - 4}$

สลับตัวแปร $x$ กับ $y$: $x = \frac{3y + 2}{y - 4}$

แก้สมการหา $y$:

$x(y - 4) = 3y + 2$

$xy - 4x = 3y + 2$

$xy - 3y = 4x + 2$

ดึงตัวร่วม $y$:

$y(x - 3) = 4x + 2$

$y = \frac{4x + 2}{x - 3}$

ดังนั้น $f^{-1}(x) = \frac{4x + 2}{x - 3}$

คำตอบ: $f^{-1}(x) = \frac{4x + 2}{x - 3}$

ข้อที่ 7

ถ้า $f(x) = x^2 + 5$ เมื่อ $x \geq 0$ จงหาค่าของ $f^{-1}(14)$

แนวคิด

ถ้า $f^{-1}(y) = x$ แล้ว $f(x) = y$

ในที่นี้ เราต้องการหา $x$ ที่ทำให้ $f(x) = 14$ และ $x \geq 0$

เฉลย

ให้ $f^{-1}(14) = x$

ตามนิยามของฟังก์ชันผกผัน จะได้ $f(x) = 14$

แทน $f(x) = x^2 + 5$ เข้าไป:

$x^2 + 5 = 14$

$x^2 = 9$

$x = \pm\sqrt{9}$

$x = \pm 3$

เนื่องจากโดเมนของ $f(x)$ คือ $x \geq 0$ ดังนั้นเรนจ์ของ $f^{-1}(x)$ จะต้อง $\geq 0$

เราจึงเลือก $x = 3$

คำตอบ: $3$

ข้อที่ 8

กำหนดให้ $f(x) = \frac{1}{x + 3}$ และ $g(x) = \sqrt{x - 1}$ จงหา $(f \circ g)(x)$ และ $D_{f \circ g}$

แนวคิด

หาโดเมนของ $f(x)$ และ $g(x)$ แยกกัน

หา $(f \circ g)(x) = f(g(x))$

กำหนดโดเมนของ $(f \circ g)(x)$ โดยที่ $x$ ต้องอยู่ในโดเมนของ $g$ และ $g(x)$ ต้องอยู่ในโดเมนของ $f$

เฉลย

โดเมนของ $f(x) = \frac{1}{x + 3}$ คือ $D_f = \{x \mid x \neq -3\}$

โดเมนของ $g(x) = \sqrt{x - 1}$ คือ $x - 1 \geq 0 \Rightarrow x \geq 1$ ($D_g = [1, \infty)$)

หา $(f \circ g)(x)$:

$(f \circ g)(x) = f(g(x)) = f(\sqrt{x - 1})$

แทน $g(x)$ เข้าไปใน $f(x)$:

$f(\sqrt{x - 1}) = \frac{1}{\sqrt{x - 1} + 3}$

หาโดเมน $D_{f \circ g}$:

  • $x$ ต้องอยู่ใน $D_g$: $x \geq 1$
  • $g(x)$ ต้องอยู่ใน $D_f$: $\sqrt{x - 1} \neq -3$

เนื่องจาก $\sqrt{x - 1} \geq 0$ เสมอ ดังนั้น $\sqrt{x - 1}$ จะไม่มีทางเท่ากับ $-3$

เงื่อนไขนี้จึงเป็นจริงเสมอ

ดังนั้น เงื่อนไขเดียวคือ $x \geq 1$

คำตอบ: $(f \circ g)(x) = \frac{1}{\sqrt{x - 1} + 3}$, $D_{f \circ g} = [1, \infty)$

ข้อที่ 9

ถ้า $f(x) = x^3 - x + 1$ จงหาค่าของ $f^{-1}(1)$

แนวคิด

ถ้า $f^{-1}(y) = x$ แล้ว $f(x) = y$

ในที่นี้ เราต้องการหา $x$ ที่ทำให้ $f(x) = 1$

เฉลย

ให้ $f^{-1}(1) = x$

ตามนิยามของฟังก์ชันผกผัน จะได้ $f(x) = 1$

แทน $f(x) = x^3 - x + 1$ เข้าไป:

$x^3 - x + 1 = 1$

$x^3 - x = 0$

ดึงตัวร่วม $x$:

$x(x^2 - 1) = 0$

$x(x - 1)(x + 1) = 0$

ดังนั้น $x = 0, x = 1, x = -1$

ข้อสังเกต: ฟังก์ชันจะมีฟังก์ชันผกผันได้ก็ต่อเมื่อฟังก์ชันนั้นเป็น ฟังก์ชันหนึ่งต่อหนึ่ง

ถ้า $f(x) = x^3 - x + 1$ ฟังก์ชันนี้ ไม่เป็นฟังก์ชันหนึ่งต่อหนึ่ง (เช่น $f(0) = 1$, $f(1) = 1$, $f(-1) = 1$)

ดังนั้น $f^{-1}(1)$ ไม่มีค่าเฉพาะ หรือ $f$ ไม่มีฟังก์ชันผกผันในโดเมนจำนวนจริงทั้งหมด

คำตอบ: ฟังก์ชัน $f(x)$ นี้ไม่เป็นฟังก์ชันหนึ่งต่อหนึ่ง ดังนั้น $f^{-1}(1)$ จึงไม่มีค่าเฉพาะ

ข้อที่ 10

กำหนดให้ $f(x) = x + 1$, $g(x) = 2x$, และ $h(x) = x - 3$ จงหาค่าของ $(h \circ g \circ f)(2)$

แนวคิด

ฟังก์ชันประกอบหลายชั้น ให้ทำงานจากข้างในสุดออกไปข้างนอก: $(h \circ g \circ f)(2) = h(g(f(2)))$

เฉลย

หา $f(2)$:

$f(2) = 2 + 1 = 3$

หา $g(f(2)) = g(3)$:

$g(3) = 2(3) = 6$

หา $h(g(f(2))) = h(6)$:

$h(6) = 6 - 3 = 3$

คำตอบ: $3$
